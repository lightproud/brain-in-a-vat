关于项目

「缸中之脑」由制作人 Light 发起，是忘却前夜官方授权的社区项目。 游戏设计内容、系统内容、资产内容归属脑缸组及其合作伙伴所有，本项目引用公开可查阅信息。

项目采用「双集群 + 人类调度器」架构：Chat 集群负责分析策划，Code 集群负责开发执行， GitHub 仓库作为「共享外脑」连接所有 AI 会话。

GitHub 仓库

AI 多会话协作开发

本项目同时验证一套可复现的 AI 协作方法论。多个 AI 会话按专岗分工，仓库作为中间层传递记忆，人类担任唯一调度器。
