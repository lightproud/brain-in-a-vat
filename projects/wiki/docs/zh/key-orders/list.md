# 钥令列表

钥令 (Key Command) 是守密人的独属战斗能力，通过银钥能量驱动。银钥能量达到1000时可释放已装备的钥令，每回合可使用1次（纯混沌编成时2次）。

::: tip 银钥觉醒 (v2.0新增)
银钥能量满时，可额外选择「银钥觉醒」将指定唤醒体的灵知觉醒卡以「保留」状态置入手中。每回合可触发1次，与钥令独立冷却。
:::
