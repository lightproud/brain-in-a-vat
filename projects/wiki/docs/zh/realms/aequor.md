# 深海界域（Aequor）

<span class="realm-badge realm-aequor">深海 Aequor</span>

## 界域特性

深海界域以**触腕系统**为核心机制，通过在回合结束时自动攻击造成持续伤害。v2.4版本界域强度评级为 **S级**，增强后为当前最强界域。

## 核心机制

- **触腕系统** — 战斗开始时生成1条触腕，深海唤醒体打出部分卡牌时除狂气值外额外获得触腕，触腕在回合结束时对前排敌人进行攻击。每次释放爆发后驱使触腕进行1次攻击
- **至纯深海** — 纯深海编成时，深海精通属性效果2倍，战斗开始时自动生成的触腕数量也变为2倍
- **触手暴击** — 触手的暴击率和暴击伤害为4只出战角色均分
- **伤害公式** — 力量:触手伤害=2:1；界域精通:狂气爆发后触手追加攻击次数=2:1

## 构筑建议

1. **触手混伤队** — 弥利亚姆+奥瑞塔+凯刻斯+墨菲（经典深海首选，攻防兼备）
2. **深海双C队** — 图鲁+奥瑞塔+希莱斯特+珊（刷本速度快）
3. **凯刻斯反击队** — 凯刻斯+哈姆林+戈利亚+拉蒙娜（顶配「凯狗哈环」，需高练度）
4. **鲨凯队** — 戈利亚+凯刻斯+希莱斯特+珊（全常驻角色，适合新手长期投资）

::: tip 新手建议
深海最明显的特点是回合结束自动追击，知道这一点之后就可以补刀收割。深海是开荒最舒服的队伍。仅需凯刻斯、法洛思、奥瑞塔、希莱斯特4张SR即可通关全主线。
:::
