# 超维界域（Ultra）

<span class="realm-badge realm-ultra">超维 Ultra</span>

## 界域特性

超维界域以**超维空间**为核心机制，通过蓄积卡牌获得额外回合。v2.4版本界域强度评级为 **A级**，老牌强队，上限高但已被深海/血肉超越。

## 核心机制

- **超维空间** — 超维界域才能发动后，队伍头上显示「超维空间」。每回合最初使用的指令卡牌的副本会蓄积其中，达到上限后进入额外的超维回合。超维回合中，超维空间中所有卡牌变为手牌
- **至纯超维** — 纯超维编成时，熟练度效果变为2倍，追加回合中的能力减少惩罚被完全无效化
- **额外回合** — 超维的机制能给队伍带来一个额外回合，能有效带来输出空间和缓冲时间

## 构筑建议

1. **超维毒队（新手推荐）** — 旺达+诺缔拉(1觉)+艾瑞卡(1觉)+宁菲亚/温柯尔（最佳新手开荒队，只有旺达是必须的，其余可替换。界域精通推荐100+）
2. **超维四小只** — 莉兹(1觉)+艾瑞卡+奥吉尔+朵尔（全免费/初始角色，叠毒生存即可。注意：当前版本莉兹毒队Meta已被超越）
3. **超维双C双盾** — 温柯尔+艾瑞卡+莉兹+弹性位（多段攻击敌人时优先温柯尔飞蛾火叠绿炎，再用艾瑞卡超限获取护盾）

::: tip 新手建议
超维强度不错但有一定上手难度。选择低算力消耗的卡牌以支撑额外回合，利用高频率行动快速积累狂气。
:::
