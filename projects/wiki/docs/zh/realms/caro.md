# 血肉界域（Caro）

<span class="realm-badge realm-caro">血肉 Caro</span>

## 界域特性

血肉界域以**胚胎系统**为核心机制，通过吞噬胚胎触发强力的爆发效果。v2.4版本界域强度评级为 **S级**，神父+专武后与深海并列。上手难度最高。

## 核心机制

- **胚胎系统** — 血肉界域才能发动后，队伍头上显示胚胎量表。量表通过血肉觉醒体的技能等积蓄（胚融合），达到100%时向手牌中加入1张「胚」卡片
- **吞噬效果** — 手中持有「胚」卡时，可与血肉角色的狂气爆发技能配合发动「吞噬」效果，消耗「胚」来触发超强力效果
- **至纯血肉** — 纯血肉编成时触发特殊增强效果
- **胚胎引擎** — 雷娅吞噬胚胎大量回血，艾继丝生产/管理胚胎。界域精通34+可自然每回合+35胚融合值

## 构筑建议

1. **尤乌哈希血肉队** — 尤乌哈希+雷娅+艾继丝+菲茵特（标准血肉，觉醒优先：艾继丝>雷娅>尤乌>菲茵特）
2. **神父队** — 萨尔瓦多+雷娅+艾继丝+24/拉蒙娜（公认Top3最强阵容之一。神父+专武后将血肉推至S级）
3. **24血肉变体** — 24+希洛/雷娅+艾继丝+萨尔瓦多（24的血肉共鸣：算力消耗=胚融合+力量增益）

::: tip 新手建议
血肉的上手难度最高，开荒时很难把控血量与胚胎的优势。雷娅+艾继丝是血肉队不可替代的「2」，优先培养。每场战斗结束时保持一个辅助能量满+胚融合70-100%。
:::
