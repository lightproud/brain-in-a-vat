# 血肉界域（Caro）

<span class="realm-badge realm-caro">血肉 Caro</span>

## 界域特性

血肉界域以**生成和消耗胚胎卡**为核心机制，通过吞噬胚胎触发强力的爆发效果。

## 核心机制

- **胚胎生成** - 血肉卡牌可以创建「胚胎」卡牌
- **吞噬/消耗** - 消耗胚胎卡触发强力效果
- **爆发循环** - 积累胚胎 → 一次性消耗 → 爆发伤害

## 构筑建议

1. 前期积累胚胎，后期一波爆发
2. 注意胚胎生成和消耗的节奏
3. 可搭配混沌界域增加前期生存能力
