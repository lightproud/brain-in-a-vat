# 混沌界域（Chaos）

<span class="realm-badge realm-chaos">混沌 Chaos</span>

## 界域特性

混沌是最灵活的界域，其核心特性是**可以与任意一个其他界域混搭**，同时加速银钥能量（Silver Key）的积累。

## 核心机制

- **界域混搭** - 混沌唤醒体可与其他界域的唤醒体组队而不损失协同效果
- **银钥加速** - 混沌卡牌在使用时会额外增加银钥能量的积累速度
- **通用性** - 适合作为任何阵容的补充

## 构筑建议

混沌界域的构筑核心在于利用其灵活性，可以：

1. 搭配深海体系加速触手铺场
2. 搭配血肉体系补充生存能力
3. 搭配超维体系提供额外资源
4. 利用银钥加速来更早发动钥令
