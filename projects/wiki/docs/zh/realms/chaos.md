# 混沌界域（Chaos）

<span class="realm-badge realm-chaos">混沌 Chaos</span>

## 界域特性

混沌是最灵活的界域，其核心特性是**可以与任意一个其他界域混搭**，同时加速银钥能量的积累。v2.4版本界域强度评级为 **A+**。

## 核心机制

- **死亡抵抗** — 任意队伍中如果存在混沌唤醒体，可以额外获得100%死亡抵抗（免死1次）
- **银钥共鸣** — 纯混沌编成时，每回合结束自动积攒250点银钥能量。每回合可使用2次钥令，同回合第2次使用钥令时，可从已解锁的其他钥令中选择一个使用
- **界域混搭** — v1.x可与任意界域角色组队；v2.0后仍保持此特性

## 构筑建议

混沌界域的构筑核心在于利用死亡抵抗和银钥共鸣：

1. **纯混沌** — 每回合自动积攒250银钥能量，可每回合使用2次钥令。推荐阵容：24+拉蒙娜+奥吉尔+莉莉
2. **搭配深海** — 深海共生提供额外触手攻击+狂气。推荐：24+图鲁+墨菲+珊
3. **搭配血肉** — 血肉共生蓄积猩红炉+胚融合加速
4. **搭配超维** — 超维共生提供生命力加成和暴伤增益

::: tip 新手建议
混沌最明显的就是死亡抵抗，知道这一点之后就敢放心卖死抗。新手开荒推荐使用初始四小只（混沌SR角色）。
:::
