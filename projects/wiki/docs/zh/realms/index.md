# 界域系统

界域（Designated Realm）是忘却前夜的核心战斗分类系统。每位唤醒体和卡牌都属于特定的界域，界域决定了卡牌的机制和战斗风格。

## 四大界域

| 界域 | 英文名 | 核心机制 | 风格 |
|------|--------|----------|------|
| <span class="realm-badge realm-chaos">混沌</span> | Chaos | 混搭其他界域，加速银钥能量 | 灵活多变 |
| <span class="realm-badge realm-aequor">深海</span> | Aequor | 召唤触手 | 持续场控 |
| <span class="realm-badge realm-caro">血肉</span> | Caro | 生成胚胎卡，吞噬/消耗 | 高爆发 |
| <span class="realm-badge realm-ultra">超维</span> | Ultra | 打出卡牌获得额外回合 | 高速连击 |

## 界域详情

- [混沌 Chaos](./chaos) - 混搭万金油
- [深海 Aequor](./aequor) - 触手召唤
- [血肉 Caro](./caro) - 胚胎吞噬
- [超维 Ultra](./ultra) - 额外回合
