# 历史活动与版本线

本页记录忘却前夜各版本更新时间线与主要内容。

## 版本更新时间线

| 版本 | 标题 | 时期 | 主要内容 |
|------|------|------|----------|
| v1.0 | 正式上线 | 2023 | 游戏正式上线；忘却篇（序章-第五章）主线；初始角色阵容；四大界域战斗系统；相位对弈PVP模式 |
| v1.1 | 1.1版本 | 2024年初 | 新角色：艾尔瓦、诺缔拉；首个限时活动「故事的魔法」；「融灾禁区」上线；「灵知深化」系统(等级上限提升至70级)；新角色：莉莉(混沌防御)；达芙黛尔角色重做 |
| v1.2 | 1.2版本 | 2024 | 融灾禁区解锁条件降低(5-4降至4-3)；幻梦深潜全造物组上线并平衡调整 |
| v1.3 | 1.3版本 | 2024 | 持续在线维护与内容优化 |
| v1.4 | 1.4版本 | 2024-2025 | 弥利亚姆增强(技能调整为穿透伤害)；多个角色加强；v1.4.4版本更新 |
| v1.5 | 1.5版本 | 2025 | 新唤醒体：奥尔拉、莱克；「超限爆发」与「超限因果」系统；现有角色免费获得新天赋；超维界域增强；减负计划；幻梦深潜翻新；多个角色复刻卡池 |
| v2.0 | 星辰正位之刻 | 2025-2026 | 全新主线「星辰篇」；编队规则大改(任意2界域混编)；钥令系统上线；混沌界域重做(银钥共鸣)；SSR出率1%->3%，保底80->30抽；大部分角色升级为SSR；R命轮完全重做 |
| v2.1 | 星辰篇第一章 | 2025 | 星辰篇第一章(1-1至1-14)；新限定：克莱门汀(超维)、克珀珊特(深海) |
| v2.2 | 星辰篇第二章「蜕变」 | 2025 | 星辰篇第二章(2-1至2-18)；新限定：凯蒂古拉(混沌)、卡斯托尔(超维,v2.2.2) |
| v2.3 | 星辰篇第三章 | 2025 | 星辰篇第三章(3-1至3-11)；新限定：茉夏(血肉)、波吕克斯(超维,v2.3.4「神圣赦免」) |
| v2.4 | 星辰篇第四章「乐园」/ 二周年 | 2025-2026 | 星辰篇第四章(4-0至4-16)，难度跃升最大章节；二周年庆典；新角色类型「合唱」；新限定：诞妄·墨菲(深海合唱)、徐(血肉合唱)；神域系统；日服预告(2025年8月19日) |
| v2.5 | 2.5版本（预告） | 2026 | 新角色阿拉克涅(Arachne)预告 |

## 重大系统变更

| 版本 | 变更 | 详情 |
|------|------|------|
| v2.0 | SSR出率提升 | SSR基础出率从1%提高到3% |
| v2.0 | 保底次数降低 | 触发保底从80抽降低到30抽 |
| v2.0 | 钥令系统上线 | 守密人专属战斗能力，消耗1000银钥能量发动 |
| v2.0 | 混沌界域重做 | 纯混沌编成时银钥共鸣：每回合自动积攒250银钥能量，可每回合使用2次钥令 |
| v2.0 | 界域混编解禁 | 任意两种不同界域可同时编队 |
| v2.0 | 命轮系统优化 | R命轮完全重做；到达一定进程后可额外装备R/SR命轮 |
| v1.5 | 超限爆发与因果 | 部分角色获得新终极技能 |
| v1.5 | 天赋系统 | 现有角色免费获得新天赋(被动能力树) |
| v1.1 | 灵知深化 | 唤醒体等级上限提升至70级 |
| v1.1 | 融灾禁区上线 | 首个多队伍终局挑战模式 |
