# 密契

密契（Covenants）是忘却前夜中的通用装备系统，不限特定唤醒体使用。密契是游戏中主要的资源消耗点之一。

## 系统概述

- 密契为通用装备，任何唤醒体都可以装备
- 密契提供属性加成和被动效果
- 密契是后期主要的培养方向

## 密契稀有度

| 稀有度 | 说明 |
|--------|------|
| <span class="rarity-ssr">SSR</span> | 最高品质密契 |
| <span class="rarity-sr">SR</span> | 高级密契 |
| <span class="rarity-r">R</span> | 基础密契 |
