# 密契列表

密契 (Covenants) 是装备套装系统，每套6件，提供3件套和6件套效果。账号35级解锁。

::: tip 主要属性词条
暴击率、暴击伤害、伤害强效、银钥充能、狂气回充、界域精通、死亡抵抗、黑印掉落
:::
