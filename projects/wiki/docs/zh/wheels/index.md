# 命轮

命轮（Wheels of Destiny）是忘却前夜中角色专属的装备系统。每位唤醒体都有对应的专属命轮。

## 系统概述

- 每位唤醒体拥有一个 **金色 SSR 专属命轮**
- 命轮通过召唤（Gacha）获取
- 命轮提供属性加成和特殊效果

## 命轮稀有度

| 稀有度 | 说明 |
|--------|------|
| <span class="rarity-ssr">SSR</span> | 角色专属金色命轮，效果最强 |
| <span class="rarity-sr">SR</span> | 高级命轮 |
| <span class="rarity-r">R</span> | 基础命轮 |
