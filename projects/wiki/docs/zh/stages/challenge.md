# 挑战关卡

挑战关卡提供高难度战斗内容和丰厚奖励。

## 融灾禁区 (Erosion Disaster Zone)

多队伍终局挑战内容，主要「黑色供物」获取来源，含稀有独占奖励。

| 属性 | 说明 |
|------|------|
| 类型 | 高难挑战 |
| 重置 | 每两周 周四15:00 |
| 解锁 | 通关主线4-3（原5-4，v1.2降低） |
| 限制 | 不可跨队伍重复使用同一唤醒体/命轮/钥令 |
| 关键奖励 | 原初灵核、灵核碎片、黑色贡物 |

### 警戒等级

| 等级 | 特训值范围 | 敌人等级 | 备注 |
|------|-----------|----------|------|
| I | 50/55/60/65/70 | 50-58 | — |
| II | 55/60/65/70/75 | 55-63 | — |
| III | 60/65/70/75/80 | 60-68 | 受融灾黑潮影响 |

### 融灾黑潮debuff

| debuff名称 | 效果 |
|------------|------|
| 狂气衰竭 | 降低狂气获取 |
| 银钥振荡 | 干扰银钥能量 |
| 怒海狂澜 | 深海界域特殊效果 |
| 猩红哺育 | 血肉界域特殊效果 |
| 彼界终途 | 超维界域特殊效果 |
| 来自遗忘 | 混沌界域特殊效果 |

### 奖励节点

特训值达到300以上，每档额外获得：黑色贡物x2500 + 蔷薇金券x100,000。III级300特训值的黑色贡物从1500提高为2500。

## 幻梦深潜 (Phantasmal Dive)

Roguelike爬塔模式，含段位系统。v1.5进行了重大翻新改造。

| 属性 | 说明 |
|------|------|
| 类型 | Roguelike排位 |
| 重置 | 每周 |
| 掉落 | 融蚀之珠(D-Jewels)、银芯 |
| 融蚀之珠公式 | (幻梦等级+2)x100 |

### Roguelike机制

- **地图** — 战争迷雾六角格地图，每日更换。节点类型：战斗/精英/Boss/商店/休息/随机事件/宝藏
- **刻印** — 击败普通敌人后获得(3选1)。优先级：妙手 > 灵感 > 算力 > 镜像伙伴 > 回想
- **造物** — 击败精英敌人后获得(3选1)。分银/金/诅咒三档。优先级：算力造物 > 狂气 > 暴击率 > 抽牌 > 触腕上限
- **黑印** — 战斗掉落，在融痕(交换点)消费。消费优先级：造物 > 觉醒 > 普通刻印

### 段位与商店

| 商店物品 | 价格 |
|----------|------|
| 追念羽笔 | 800融蚀之珠=5/周 |
| 碎裂余梦 | 100融蚀之珠/周 |
| 群星余梦 | 25融蚀之珠/周 |

## 相位对弈 (Phase Chess)

卡牌PVP对战，仍在Beta测试中。角色练度不影响PVP——所有卡牌以相同强度解锁。

| 属性 | 说明 |
|------|------|
| 类型 | PVP排位 |
| 重置 | 每周 |
| 解锁 | 通关主线3-2 |
| 掉落 | 演习徽章、蔷薇金券 |

### PVP规则

| 规则 | PVE | PVP |
|------|-----|-----|
| 初始手牌 | — | 5张 |
| 换牌 | — | 可自由选择任意张数更换 |
| 每回合抽牌 | 4张 | 5张 |
| 手牌保留 | 回合末弃牌 | 手牌保留至下回合 |
| 初始算力 | 固定5点 | 3点起每回合+1至上限10 |
| 角色练度 | 有效 | 所有卡牌等强度 |
| 阵亡角色卡 | 移除 | 变为「泪滴」(0费，恢复10点能量) |

### 死斗机制

双方算力上限均达到10后自动施加死斗效果：先降低治疗和护盾效果，再将所有治疗和护盾变为1点，强制分出胜负。

### 段位

新手段位 → 资深I → 传说 → 相位传奇(最高段)
