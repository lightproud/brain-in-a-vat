# 资源关卡

资源关卡是获取角色培养材料的主要途径。

## 资源副本一览

### 融蚀之墟 (Dissolution Ruins)

主要素材获取地点，消耗墨诺芬（体力）。每日重置。

**解锁条件：** 通关调查行动1-2

| 副本类型 | 掉落内容 |
|----------|----------|
| 升格素材 | 初阶/中阶/高阶升格匣子（各界域专属变体） |
| 技能素材 | 初阶/中阶/高阶技能补给 |
| 界域残渣 | 混沌残渣、深渊残渣、血腥残渣、宇宙残渣 |
| 教化素材 | 苍白笔记、苍白卷宗、苍白典籍 |
| 通用掉落 | 蔷薇金券、灵知制剂、突破材料 |

### 禁忌篆录 (Forbidden Inscriptions)

每日重置的素材副本。通关后解锁重复挑战。

### 超验存在 (Transcendent Existence)

| 类型 | 重置时间 | 说明 |
|------|----------|------|
| 周Boss | 每周一15:00 | 奖励丰厚，建议每周必清 |

## 每日/每周系统

| 系统 | 重置 | 奖励 | 说明 |
|------|------|------|------|
| 每日试训 (Daily Training) | 每日 | 500经验/天 | 由原每日+每周系统合并而成。可使用代理凭证快速完成（上限5次） |
| 每周试炼 (Weekly Trial) | 每周 | 银芯 100/150/250 + 融蚀之珠 | 每周挑战任务 |
| 外勤代理 (Assignments) | 每日结算 | 蔷薇金券、融蚀之珠、各类素材 | 自动派遣系统。研究任务可获4000基础+每20等级差额外2000蔷薇金券 |
| 课程 (Curriculum) | — | 蔷薇金券、拟态凝晶、千面印章 | 进阶课题获取拟态凝晶等核心素材 |

## 素材获取途径汇总

| 素材类型 | 主要来源 |
|----------|----------|
| 升格匣子 | 融蚀之墟(升格素材副本) |
| 技能补给 | 融蚀之墟(技能素材副本) |
| 界域残渣 | 融蚀之墟(对应界域副本) |
| 蔷薇金券 | 主线首通、融蚀之墟、外勤代理、课程 |
| 灵知制剂 | 融蚀之墟、活动奖励 |
| 拟态凝晶 | 课程进阶、商店 |
| 追念羽笔 | 各商店(55级后解锁，每周25-50个) |

::: tip
三星通关所有关卡可累计获得约4750银芯（每关50银芯）。高难度通关自动解锁低难度三星。
:::
