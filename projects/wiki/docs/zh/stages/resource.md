# 资源关卡

资源关卡是获取角色培养材料的主要途径。

## 资源关卡类型

| 关卡类型 | 掉落内容 | 开放时间 |
|----------|----------|----------|
| 经验本 | 角色经验材料 | 每日开放 |
| 金币本 | 游戏货币 | 每日开放 |
| 属性材料本 | 角色突破材料 | 轮换开放 |
| 技能本 | 技能升级材料 | 轮换开放 |
| 武器本 | 武器强化材料 | 轮换开放 |
| 装备本 | 装备掉落 | 每日开放 |
