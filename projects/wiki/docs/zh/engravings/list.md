# 刻印列表

刻印（Engravings）是为唤醒体提供额外增益的系统。

## 刻印类型

| 刻印名称 | 效果分类 | 说明 |
|----------|----------|------|
| 攻击刻印 | 攻击提升 | 提升基础攻击力 |
| 防御刻印 | 防御提升 | 提升基础防御力 |
| 生命刻印 | 生命提升 | 提升最大生命值 |
| 精通刻印 | 界域精通 | 提升界域精通属性 |
| 暴击刻印 | 暴击相关 | 提升暴击率或暴击伤害 |

## 获取途径

- 关卡掉落
- 活动奖励
- 商店兑换
