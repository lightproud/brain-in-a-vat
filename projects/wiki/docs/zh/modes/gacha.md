## 保底机制

| 保底类型 | 说明 |
|----------|------|
| 小保底 | **30抽** 必出SSR（v2.0后，原v1.x为80抽） |
| UP概率 | 抽到SSR时，33.33%概率为UP角色/命轮 |
| 递增保底 | 第1次SSR若歪(非UP)，第2次SSR有66.66%概率为UP；若仍歪，第3次100%为UP |
| 大保底 | **120抽** 内必获UP角色（4个30抽周期内） |
| 防重复 | 连续两次获得相同UP物品后，下次UP必为另一个UP物品 |
| 跨池独立 | 限定池与常驻池保底独立计算，各自维护计数器 |

## 新手福利

- 开服赠送1500银芯(约7抽) + 普池15抽
- 累计登录7天额外送10抽活动池
- 达到35级时最高可获40抽普池 + 2000银芯(10抽)
- 累计登录3天、7天各送1次常驻自选SSR
- 累计抽卡50次送1次常驻自选SSR
- 7天后至少拥有3只以上SSR角色
- 每周轮换唤醒活动首5抽免费(活动期间最多20抽免费)

## 抽卡建议

1. **普池先下角色池** - 只用送的无垢之芯
2. **抽齐核心角色收手** - 不要过度追求满命
3. **剩余无垢下武器池** - 拿专武提升战力
4. **银芯和流明只冲限定池** - 限定角色过期后难以获取
5. **限定池保底不跨池共享** - 限定与常驻独立计算
6. **F2P优先确保24** - 其次墨菲，两者为最强投资
