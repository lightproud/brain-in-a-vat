# 战斗系统

忘却前夜采用 **回合制卡牌战斗** 系统，在 2D 横版视角下进行。

## 战斗基础

### 回合流程

1. **抽牌阶段** - 从卡组中抽取卡牌到手牌
2. **行动阶段** - 消耗算力打出卡牌
3. **结束阶段** - 触发回合结束效果，进入敌方回合

### 算力系统（Arithmetica）

| 属性 | 数值 |
|------|------|
| 每回合基础算力 | 5 点 |
| 卡牌消耗范围 | 1~5 算力不等 |
| 算力上限 | 可通过效果提升 |

算力是打出卡牌的资源，合理管理算力是战斗胜利的关键。

### 狂气系统（Madness Points）

- 通过使用卡牌和受击等方式积累狂气
- 狂气积累至 **100 点** 时触发 **狂气爆发（Madness Burst）**
- 狂气爆发不消耗算力，效果极其强力
- 合理控制狂气积累节奏是进阶技巧

### 银钥系统（Silver Key）

- 使用卡牌会积累银钥能量
- 银钥能量达到 **1000 点** 时可发动钥令
- 钥令在战前装备，代表守密人的力量
- 混沌界域可加速银钥能量积累

## 卡牌类型

| 类型 | 说明 |
|------|------|
| <span class="card-type-badge card-command">指令卡</span> | 每位唤醒体 4 张，消耗算力使用 |
| <span class="card-type-badge card-spirit">灵魂觉醒卡</span> | 每位唤醒体 1 张，强力觉醒技能 |
| <span class="card-type-badge card-madness">狂气爆发</span> | 狂气满 100 触发，不消耗算力 |

## 队伍编成

- 最多 **4 名唤醒体** 组成队伍
- 可装备一个钥令
- 建议阵容考虑界域协同

## 战斗技巧

1. **算力管理** - 不要每回合都用光算力，留些余量应对突发
2. **狂气时机** - 在关键时刻触发狂气爆发，最大化收益
3. **界域联动** - 利用同界域卡牌的协同效果
4. **银钥规划** - 针对不同敌人选择合适的钥令
5. **手牌管理** - 注意手牌上限，合理使用或丢弃卡牌
