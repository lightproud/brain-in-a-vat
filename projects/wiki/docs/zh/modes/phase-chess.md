# 相位对弈（PvP）

相位对弈是忘却前夜的 PvP 对战模式，强调纯粹的卡牌构筑技巧。

## 解锁条件

通关主线 **3-2** 关卡后解锁。

## 基本规则

| 规则 | 数值 |
|------|------|
| 初始能量 | 3 |
| 每回合能量增加 | +1 |
| 最大能量 | 10 |
| 每回合抽牌 | 5 张 |
| 手牌上限 | 10 张 |

## 公平性

相位对弈的核心特点是 **完全公平**：

- **无角色数值差异** - 所有角色的属性在PvP中统一
- **无装备影响** - 命轮、密契等装备效果不生效
- **纯构筑竞技** - 完全依靠卡牌构筑和操作技巧

## 竞技建议

1. **适应能量曲线** - 前期能量少，打低费卡；后期能量多，打高费combo
2. **针对性构筑** - 了解当前热门构筑，准备应对方案
3. **手牌管理** - 注意10张手牌上限，避免爆牌
4. **节奏把控** - 在合适时机发动攻势
