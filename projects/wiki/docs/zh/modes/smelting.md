# 冶炼室

冶炼室（Smelting Room）是忘却前夜中的道具制作和材料转换系统。

## 系统概述

在冶炼室中可以使用材料制作各种道具和造物。这是获取特定材料和升级道具的重要途径。

## 可制作物品

| 类别 | 说明 |
|------|------|
| 造物 | 可装备或使用的制作道具 |
| 消耗品 | 战斗中或冒险中使用的道具 |
| 材料转换 | 将低级材料合成为高级材料 |

## 建议

- 关注每日免费刷新的制作配方
- 优先制作当前队伍需要的升级材料
- 材料转换可帮助消耗多余的低级材料
