# 融灾禁区

融灾禁区（Erosion Disaster Zone / Meltdown Zone）是忘却前夜的多队伍终局挑战模式。

## 模式概述

融灾禁区是对玩家角色池深度的终极考验。需要组建多支队伍，使用不重复的唤醒体、命轮和钥令通关五个连续挑战关卡。

## 规则

- **多队伍挑战**：构建多支不同队伍——所有队伍之间唤醒体、命轮、钥令不可重复使用
- **角色池深度**：成功取决于拥有足够多已培养的角色
- **策略规划**：所有队伍的阵容搭配比单一队伍更重要

## 攻略建议

- 将最强唤醒体分散到各队伍中，而非集中在一队
- 开始前提前规划命轮和钥令的分配
- 考虑哪些关卡最适合特定界域的协同效果
- 建议至少有2-3支完全投入的队伍后再尝试
