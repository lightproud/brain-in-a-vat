# 游戏模式

忘却前夜包含多种游戏模式，涵盖 PvE 战斗、Roguelite 探索和 PvP 对战。

## 模式一览

| 模式 | 类型 | 说明 |
|------|------|------|
| [战斗系统](./combat) | 核心 | 回合制卡牌战斗系统详解 |
| [调查行动](./investigation) | PvE Roguelite | 六角格地图探索模式 |
| [相位对弈](./phase-chess) | PvP | 公平卡牌对战，无数值差异 |
| [意识潜游](./consciousness-dive) | PvE | 特殊挑战模式 |
| [异梦视界](./dream-visions) | PvE | 特殊挑战模式 |
| [繁衍狂热](./proliferation) | PvE | 特殊挑战模式 |
| [召唤系统](./gacha) | 抽卡 | 获取唤醒体和命轮 |
| [冶炼室](./smelting) | 制作 | 道具制作系统 |
| [派遣](./dispatch) | 探险 | 派遣唤醒体获取资源 |
