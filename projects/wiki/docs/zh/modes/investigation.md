# 调查行动

调查行动是忘却前夜的核心 PvE Roguelite 模式。玩家在六角格地图上探索，遭遇战斗、事件和商店。

## 基本规则

- 最多 **4 名唤醒体** 组队进入调查
- 六角格（Hex）地图探索
- 地图布局非随机，但事件/掉落/商店内容随机
- 每次调查包含多个区域和Boss战

## 地格类型

| 地格 | 图标 | 说明 |
|------|------|------|
| 敌人遭遇 | ⚔️ | 进入战斗 |
| 事件 | ❓ | 触发随机事件 |
| 隐藏银 | 💰 | 获取银币资源 |
| 黑色印记 | 🔮 | 特殊效果 |
| 交换 | 🔄 | 交换道具/卡牌 |
| 方尖碑 | 🗿 | 提供增益效果 |
| 商店 | 🏪 | 购买道具/卡牌 |

## 关卡助力

调查行动中可获得临时增益（关卡助力），在本次调查中生效。

## 攻略建议

1. **合理规划路线** - 优先前往事件和商店地格获取资源
2. **适时战斗** - 战斗可获取经验和掉落
3. **资源管理** - 合理分配银币用于商店购买
4. **关卡助力选择** - 选择与当前构筑方向匹配的助力
