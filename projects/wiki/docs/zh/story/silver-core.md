# 银芯通信

银芯通信是忘却前夜中的唤醒体好感度/羁绊系统。

## 系统概述

通过银芯通信，守密人（玩家）可以与唤醒体建立更深的羁绊关系，解锁专属剧情、语音和其他奖励。

「银芯」是一种通信装置，让守密人能够与唤醒体的心灵和记忆建立连结。这一系统与游戏「记忆与遗忘」的核心主题紧密相连——通过银芯通信，玩家不仅能提升角色实力，更能深入了解每位唤醒体背后的故事与秘密。

## 好感度提升方式

- **赠送礼物** -- 赠送唤醒体喜欢的物品，可获得大量好感度。不同唤醒体有各自的喜好偏向，中性礼物仅获少量好感，不喜欢的礼物可能无收益
- **互动对话** -- 通过银芯通信界面与唤醒体进行日常互动，每日积少成多
- **编入队伍** -- 将唤醒体编入战斗队伍出战，通过共同作战经历缓慢积累好感
- **完成任务** -- 完成与唤醒体背景故事相关的专属任务，可获得大量好感度提升

## 解锁内容

| 好感度等级 | 可解锁内容 |
|-----------|-----------|
| 1 | 基础互动对话 |
| 2 | 第一段专属剧情 |
| 3 | 新增语音台词 |
| 4 | 属性加成（少量） |
| 5 | 第二段专属剧情 |
| 6 | 额外语音台词 |
| 7 | 属性加成（中等） |
| 8 | 第三段专属剧情 |
| 9 | 特殊外观内容（表情、资料装饰等） |
| 10 | 最终专属剧情及满级属性加成 |

::: info
各等级所需的具体好感度点数会随版本更新调整。高等级所需点数远高于低等级。
:::
