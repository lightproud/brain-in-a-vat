# 世界观设定

## 世界背景

忘却前夜的世界观深受 **克苏鲁神话（Cthulhu Mythos）** 影响。故事发生在一个充满未知恐惧的世界中，「忘却之夜」是一种周期性出现的超自然现象。

## 核心概念

| 概念 | 说明 |
|------|------|
| 忘却之夜 | 周期性出现的神秘现象 |
| 唤醒体（Awakener） | 被「忘却之夜」唤醒特殊能力的个体 |
| 守密人（Keeper） | 玩家所扮演的角色 |
| 界域（Designated Realm） | 四种不同的力量分类 |
| 算力（Arithmetica） | 战斗中使用卡牌的能量 |
| 狂气（Madness） | 与克苏鲁神话中「疯狂」概念对应的力量 |
| 银钥（Silver Key） | 洛夫克拉夫特作品中的银钥意象 |

## 四大界域

| 界域 | 灵感来源 | 特征 |
|------|----------|------|
| 混沌（Chaos） | 阿撒托斯等混沌之神 | 无序、混搭、灵活 |
| 深海（Aequor） | 克苏鲁、深潜者 | 海洋、触手、深渊 |
| 血肉（Caro） | 奈亚拉托提普等 | 肉体、变异、吞噬 |
| 超维（Ultra） | 高维存在 | 时空、维度、超越 |
