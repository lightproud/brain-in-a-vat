# 主线剧情

::: warning 剧透警告
以下内容包含游戏主线剧情的剧透，请酌情阅读。
:::

忘却前夜主线剧情总文本量约80万字，采用单元剧形式，分为「忘却篇」和「星辰篇」两大篇章。
