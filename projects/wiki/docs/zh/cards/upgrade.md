# 卡牌升级

## 升级系统

卡牌可以升级最多 **6 次**，每次升级都会增强卡牌效果。

| 等级 | 效果提升 | 所需材料 |
|------|----------|----------|
| Lv.1 | 基础效果 | - |
| Lv.2 | 效果增强 | *待补充* |
| Lv.3 | 效果增强 | *待补充* |
| Lv.4 | 效果增强 | *待补充* |
| Lv.5 | 效果增强 | *待补充* |
| Lv.6 | 最大效果 | *待补充* |

## 升级优先级建议

1. **核心输出卡** - 优先升级主力唤醒体的主要伤害卡牌
2. **关键控制卡** - 提供重要控制/增益效果的卡牌
3. **灵魂觉醒卡** - 提升觉醒技能的效果
4. **辅助卡** - 最后升级辅助性质的卡牌
