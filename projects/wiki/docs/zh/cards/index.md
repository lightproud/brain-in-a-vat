# 卡牌系统

卡牌是忘却前夜战斗系统的核心。游戏拥有 185+ 张卡牌，分属四大界域。

## 卡牌类型

### 指令卡（Command Card）

每位唤醒体拥有 4 张指令卡，是战斗中最常使用的卡牌。打出指令卡需要消耗 **算力（Arithmetica）**。

### 灵魂觉醒卡（Spirit Awakening Card）

每位唤醒体拥有 1 张灵魂觉醒卡，效果强力。

### 狂气爆发

当 **狂气（Madness Points）** 积累至 100 点时，可触发狂气爆发，释放强力效果，且不消耗算力。

## 算力系统

| 属性 | 说明 |
|------|------|
| 每回合基础算力 | 5 点 |
| 卡牌消耗 | 因卡牌而异（1~5 点不等） |
| 算力回复 | 部分效果可额外提供算力 |
