# 物资与货币

## 付费货币

| 货币名称 | 用途 | 主要获取途径 |
|----------|------|-------------|
| 源液 (Silver Prime) | 购买抽卡、通行证、外观和礼盒 | 真金白银充值（标准档位：6/30/68/128/198/328/648元等价） |

## 抽卡货币

| 货币名称 | 用途 | 主要获取途径 |
|----------|------|-------------|
| 银芯 (Silver Core) | 主要免费高级货币，兑换流明之芯和无垢之芯；恢复墨诺芬(50银芯=120墨诺芬) | 每日任务、主线评分(100/关)、主线成就、手册、幻梦深潜、活动奖励、通行证、兑换码、地图探索拾取 |
| 流明之芯 (Lumen Core) | 限定池抽卡道具，1个=1抽 | 银芯兑换、活动奖励、通行证（获取途径少于无垢之芯，应优先兑换） |
| 无垢之芯 (Pure Core) | 常驻池/命轮池抽卡道具，1个=1抽 | 银芯兑换、活动奖励、角色首次升格、纪行免费轨道(40个)、章节首通 |

## 兑换货币

| 货币名称 | 用途 | 主要获取途径 |
|----------|------|-------------|
| 蔷薇金券 (Rose Scrip) | 蔷薇金券商店、教化唤醒体、炼金、素材合成、命轮叠加、密契誊抄(7500/次) | 主线首次通关、融蚀之墟、外勤代理、课程、拆解命轮(蓝=1000,紫=4000) |
| 沉淀币 (Sediment Currency) | 沉淀商店：密契(猩红之拥)、追念羽笔(80沉淀=10笔/周)、原初灵核(25碎片=1核,3次/月) | 抽卡(唤醒)副产物 |
| 调查徽章 (Investigation Badge) | 调查商店：追念羽笔(800徽章=5笔/周) | 主线三星通关、秘术统御、灵魂潜行三星、真实掌握(1000)、个人成就(300/个) |
| 无光币 (Lightless Currency) | 无光商店：密契(远方的欢宴)、追念羽笔(800=5/周) | 无光秘境、界域异效区 |
| 融蚀之珠 (D-Jewels) | 深潜商店：追念羽笔(800=5/周)、碎裂余梦(100/周)、群星余梦(25/周) | 幻梦深潜(三级=500,疯狂=900)、外勤代理(每日结算)。公式：(幻梦等级+2)x100 |
| 相位币 (Phase Coin) | 相位商店：密契(猩红之悸)、追念羽笔(800=5/周) | 相位对弈(PVP) |

## 体力

| 货币名称 | 用途 | 主要获取途径 |
|----------|------|-------------|
| 墨诺芬 (Menophine) | 体力/行动力资源，进入幕间演习消耗 | 自然恢复、银芯兑换(50银芯=120墨诺芬)、道具/礼盒 |

## 弥萨格商店

弥萨格商店是游戏内的商店系统，可使用各类货币兑换道具。

| 商店类别 | 货币 | 可兑换内容 |
|----------|------|-----------|
| 蔷薇金券商店 | 蔷薇金券 | 密契(荒原狼)、教化素材、各类培养材料 |
| 沉淀商店 | 沉淀币 | 密契(猩红之拥)、追念羽笔、原初灵核碎片 |
| 调查商店 | 调查徽章 | 追念羽笔、密契(猩红之悸) |
| 无光商店 | 无光币 | 密契(远方的欢宴)、追念羽笔 |
| 深潜商店 | 融蚀之珠 | 密契(诅咒兔)、追念羽笔、碎裂余梦、群星余梦 |
| 相位商店 | 相位币 | 密契(猩红之悸)、追念羽笔 |
