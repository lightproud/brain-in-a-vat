# 唤醒体图鉴

唤醒体（Awakener）是忘却前夜中的可操作角色。每位唤醒体拥有独特的属性、指令卡组和灵魂觉醒卡。

## 唤醒体分类

### 按稀有度

| 稀有度 | 说明 |
|--------|------|
| <span class="rarity-ssr">SSR</span> | 最高稀有度，通过限定召唤获取 |
| <span class="rarity-sr">SR</span> | 高稀有度，可通过召唤获取 |
| <span class="rarity-r">R</span> | 基础稀有度，较容易获取 |

### 按界域

| 界域 | 说明 |
|------|------|
| <span class="realm-badge realm-chaos">混沌 Chaos</span> | 可混搭其他界域，加速银钥能量积累 |
| <span class="realm-badge realm-aequor">深海 Aequor</span> | 召唤触手进行攻击 |
| <span class="realm-badge realm-caro">血肉 Caro</span> | 生成胚胎卡进行吞噬/消耗 |
| <span class="realm-badge realm-ultra">超维 Ultra</span> | 打出卡牌可获得额外回合 |

### 按定位

| 定位 | 说明 |
|------|------|
| 输出型（DPS） | 主要负责造成伤害 |
| 防御型（Tank） | 负责承受伤害，提供护盾 |
| 辅助型（Support） | 提供增益、治疗等辅助效果 |

### 基础属性

每位唤醒体拥有以下基础属性：

- **体质（Constitution）** - 生命值上限
- **攻击（Attack）** - 影响卡牌伤害
- **防御（Defense）** - 减少受到的伤害

### 卡牌构成

每位唤醒体拥有：
- **4 张指令卡（Command Cards）** - 日常使用的战斗卡牌
- **1 张灵魂觉醒卡（Spirit Awakening Card）** - 强力的觉醒技能
