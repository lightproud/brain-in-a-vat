# 唤醒体培养指南

## 等级提升

唤醒体可通过消耗经验材料提升等级，等级提升会增加体质、攻击和防御属性。

### 经验材料

| 材料名称 | 提供经验 | 获取途径 |
|----------|----------|----------|
| *待补充* | - | - |

## 技能升级

唤醒体的技能在达到 25 级后解锁升级功能。

### 技能升级材料

| 材料名称 | 用途 | 获取途径 |
|----------|------|----------|
| *待补充* | - | - |

## 培养优先级建议

### 通用建议

1. **优先培养主力输出唤醒体** - 提升等级和技能
2. **投资核心卡牌升级** - 提升主要输出/控制卡牌
3. **配置命轮和密契** - 选择适合定位的装备
4. **利用启灵强化** - 重复获取的唤醒体可用于启灵

::: tip
具体唤醒体的培养优先级请参考各唤醒体详情页和[节奏榜](/guides/tier-list)。
:::
