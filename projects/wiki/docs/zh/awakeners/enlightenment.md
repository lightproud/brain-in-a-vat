# 启灵系统

## 系统概述

启灵（Enlightenment）是忘却前夜中通过获取重复唤醒体来强化角色的系统，类似于其他游戏中的「命座」系统。

## 启灵效果

每次启灵会解锁不同的增强效果，包括但不限于：

- **算力消耗降低** - 减少卡牌的算力费用
- **属性增强** - 增加攻击/防御/体质加成
- **卡牌回收** - 特定条件下回收已使用的卡牌
- **技能强化** - 提升技能效果/倍率
- **特殊机制** - 解锁独特的战斗机制

## 启灵等级

| 启灵等级 | 所需重复 | 效果类型 |
|----------|----------|----------|
| 启灵 1 | 1次重复 | *因唤醒体而异* |
| 启灵 2 | 2次重复 | *因唤醒体而异* |
| 启灵 3 | 3次重复 | *因唤醒体而异* |
| 启灵 4 | 4次重复 | *因唤醒体而异* |
| 启灵 5 | 5次重复 | *因唤醒体而异* |

::: tip
部分唤醒体的关键启灵节点（如启灵1或启灵3）可能带来质变级的提升，建议抽卡前参考具体启灵效果。
:::
