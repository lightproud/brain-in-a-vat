# 阵容搭配指南

## 基本原则

忘却前夜的阵容最多包含 **4 名唤醒体**，搭配时需考虑：

1. **界域协同** - 同界域唤醒体可以更好地配合卡牌效果
2. **混沌的灵活性** - 混沌界域可以与任意一个其他界域混搭
3. **定位互补** - 输出 + 防御/辅助的均衡搭配
4. **卡牌联动** - 不同唤醒体的卡牌之间的配合
5. **钥令选择** - 根据阵容选择合适的钥令

## 界域构筑思路

### 深海（Aequor）体系

- **核心机制**: 召唤触手造成持续伤害
- **特点**: 场控能力强，适合持久战

### 血肉（Caro）体系

- **核心机制**: 生成胚胎卡并通过吞噬/消耗触发效果
- **特点**: 爆发伤害高，需要构筑配合

### 超维（Ultra）体系

- **核心机制**: 打出卡牌获得额外回合
- **特点**: 高速连击，动作频率高

### 混沌（Chaos）混搭

- **核心机制**: 混搭其他界域，加速银钥能量积累
- **特点**: 灵活多变，适合搭配不同体系

## 核心联携

| 联携名 | 角色 | 机制 |
|--------|------|------|
| 触手叠加 | 图鲁 + 奥瑞塔 + 弥利亚姆 | 各自增加触手实例，自动跟随攻击乘法叠加 |
| 胚胎引擎 | 雷娅 + 艾继丝 | 雷娅吞噬胚胎大量回血，艾继丝生产/管理胚胎 |
| 神父血池循环 | 萨尔瓦多 + 雷娅 | 雷娅复制神父2费卡，HP吸收→血池饮用→高HP=高伤害 |
| 反击缩放 | 凯刻斯 + 诺缔拉 | 凯刻斯3觉后防御卡造成反击暴击伤害，诺缔拉填充蓄能条 |
| 24万能适配 | 24 + 任意界域 | 共鸣随队伍界域变化，始终提供对应增益 |
| 护盾打击 | 珊 + 诺缔拉 | 珊的护盾型DPS需要诺缔拉的辅助支撑才能运转 |
