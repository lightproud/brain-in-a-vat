# 銀芯通信（Silver Core）

**銀芯通信**はゲーム内で閲覧できる読み物・設定資料集です。世界観の深い理解に役立つテキストが収録されています。

## 銀芯通信とは

- ゲーム内のアーカイブシステム
- メインストーリー進行やイベントクリアで新しい記事が解放
- 世界観・キャラクター・組織の背景設定が記載
