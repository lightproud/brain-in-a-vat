# ストーリー概要 — クトゥルフ神話テーマ

忘却前夜（Morimens）は**クトゥルフ神話（ラヴクラフト神話）**を世界観の基盤とした物語です。プレイヤーは覚醒体たちと共に、宇宙的恐怖（コズミックホラー）に満ちた世界で真実を追い求めます。

## 世界設定

人類の文明が発展する裏側で、太古より眠る旧支配者たちの力が目覚めつつあります。「覚醒体」と呼ばれる特殊な能力を持つ者たちは、界域の力を宿し、迫りくる忘却の夜に抗います。

## 4つの界域と神話的背景

| 界域 | 神話的モチーフ |
|:---:|:---|
| <span class="realm-badge realm-chaos">混沌</span> | ナイアルラトホテプ、アザトース — 狂気と混沌の外なる神 |
| <span class="realm-badge realm-aequor">深海</span> | クトゥルフ、ダゴン — 深海に眠る旧支配者 |
| <span class="realm-badge realm-caro">血肉</span> | シュブ＝ニグラス — 豊穣と肉体変容の暗黒母神 |
| <span class="realm-badge realm-ultra">超維</span> | ヨグ＝ソトース — 時空を超越する全知の存在 |
