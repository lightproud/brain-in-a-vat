# メインストーリー

忘却前夜のメインストーリーのあらすじと章解説です。総テキスト量は約80万字で、単元劇形式で展開されます。

::: warning
以下の内容にはストーリーのネタバレが含まれます。未読の方はご注意ください。
:::

## 忘却篇（Oblivion Chapter）

### プロローグ

守密人の旅が始まる序章。弥萨格大学に到着した主人公は、融蝕という見えない災厄の存在を知る。記憶を失った原因を調査するため、覚醒体たちと共に未知の真相へと踏み出す。

> 「弥萨格大学へようこそ、守密人。だが祝うことは何もない――あなたは誰にも知られず、誰にも称えられず、誰にも記憶されない。」

### 第1章：東区秘事（East District Secrets）

| 節 | タイトル |
|:---:|:---|
| 1-1 | 漆黒之夢 (Dream of Darkness) |
| 1-2 | 人不可貌相 (Don't Judge By Appearance) |
| 1-3 | 地下の幽霊 (Underground Phantom) |
| 1-4 | 小小偵探 (Little Detective) |
| 1-5 | 病 (Illness) |
| 1-6 | 非礼勿言 (Speak No Evil) |
| 1-7 | 誰が殺した知更鳥 (Who Killed the Robin?) |
| 1-8 | 内在的危険 (Inner Danger) |
| 1-9 | 我々の秘密 (Our Secret) |
| 1-10 | 甘美な芸術 (The Sweet Art) |

調査は層を重ねて深まっていく。まず工坊の管理者、次に怪しい医者、そして邪教組織の疑いへ。最終的な真相の核心には、一人のコントロールされた哀れな少女がいた。彼女は家族と仲間を守るため、自ら命を絶つことを選んだ。

### 第2章：蜕変（Metamorphosis）

メインストーリー第二章。融蝕の脅威がさらに深刻になり、守密人は新たな覚醒体たちと出会いながら、世界の裏側に潜む秘密に迫る。

### 第3章

メインストーリー第三章。物語の核心に近づくにつれ、組織間の対立と陰謀が明らかになっていく。

### 第4章

重大な転換点。角色たちが救い出した少女・莉莉を療養院に連れて行くが、そこでさらに大きな危機と秘密が潜んでいることを発見する。プレイヤーの選択が後続のストーリー展開に影響する。

### 第5章

メインストーリー第五章。一度閉鎖された後に再開放され、補償報酬が配布された。

### 第6章

メインストーリー第六章。調査行動の章節報酬として特批薬剤が獲得可能（第6章=35個）。

### 第7章

ボス章節を含むメインストーリー第七章。7-18「火種(Kindling)」等のボス戦が展開される。

## 星辰篇（Stellar Chapter）

v2.0アップデートで追加された新章「星辰正位之刻」。編成ルールの大改革（任意2界域混編解禁）、鍵令システムの実装、混沌界域のリメイクなど、ゲームプレイにも大きな変革をもたらした。

### 星辰篇序章

星辰篇の幕開け。新たな時代の始まりと、これまでの物語を引き継ぐ新章の導入。

### 星辰篇第1章（1-1〜1-14）

星辰正位之刻の第一章。新限定キャラクター・克莱门汀（超維）と克珀珊特（深海）が登場。

### 星辰篇第2章：蜕変（2-1〜2-18）

星辰篇第二章。新限定キャラクター・凯蒂古拉（混沌）とカ斯托尔（超維）が登場。

### 星辰篇第3章（3-1〜3-11）

星辰篇第三章。新限定キャラクター・茉夏（血肉）と波吕克斯（超維）が登場。

### 星辰篇第4章：楽園（Paradise）（4-0〜4-16）

> 「なぜ黒い太陽が空を覆ったのか？」

星辰篇で最も難易度が跳ね上がる章。v2.4二周年版本で実装。新キャラタイプ「合唱（Chorus）」が登場し、诞妄·墨菲（深海合唱型）と徐（血肉合唱型）が実装された。

## 後半ストーリーの特殊メカニクス

### 融災黒潮（Erosion Black Tide）

後半章節で出現する特殊なデバフメカニクス。以下の効果が含まれる：

| デバフ名 | 効果 |
|:---|:---|
| 狂気衰竭 | 狂気獲得量を低下 |
| 銀鍵振盪 | 銀鍵エネルギーを妨害 |
| 怒海狂瀾 | 深海界域特殊効果 |
| 猩紅哺育 | 血肉界域特殊効果 |
| 彼界終途 | 超維界域特殊効果 |
| 来自遺忘 | 混沌界域特殊効果 |
