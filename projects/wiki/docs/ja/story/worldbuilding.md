# 世界観設定

忘却前夜（Morimens）の世界観はクトゥルフ神話をベースに独自の設定が組み込まれています。

## 基本設定

忘却前夜の世界では、太古より存在する4つの界域の力が人類の歴史に影を落としています。界域の力に「覚醒」した者たちは覚醒体と呼ばれ、迫りくる宇宙的脅威に対抗する使命を負います。

## 界域と旧支配者

4つの界域はそれぞれ異なる旧支配者・外なる神の影響下にあります。

| 界域 | 関連する神話存在 | 影響 |
|:---:|:---|:---|
| <span class="realm-badge realm-chaos">混沌</span> | ナイアルラトホテプ、アザトース | 狂気・精神汚染 |
| <span class="realm-badge realm-aequor">深海</span> | クトゥルフ、ダゴン | 深海の呼び声・変異 |
| <span class="realm-badge realm-caro">血肉</span> | シュブ＝ニグラス | 肉体変容・増殖 |
| <span class="realm-badge realm-ultra">超維</span> | ヨグ＝ソトース | 次元干渉・時空歪曲 |

## 重要用語

| 用語 | 説明 |
|:---|:---|
| 覚醒体 | 界域の力に覚醒した人間 |
| 忘却前夜 | 全ての記憶・存在が消滅する「忘却」の前夜 |
| 銀鍵 | ヨグ＝ソトースの門を開く鍵。ゲーム内では戦闘リソース |
| 算力 | 覚醒体が界域の力を行使するためのエネルギー |
| 狂気 | 界域の力を使いすぎることで蓄積する精神的負荷 |
