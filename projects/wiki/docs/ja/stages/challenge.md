チャレンジステージは高難度の常設ステージです。高レアリティの密契・刻印・特殊素材がドロップします。

## 融災禁区（詳細）

多チーム終局コンテンツ。同一覚醒体・運命の輪・鍵令をチーム間で重複使用不可。

### アラートレベル

| レベル | 訓練値 | 敵レベル | 備考 |
|:---:|:---|:---:|:---|
| I | 50/55/60/65/70 | 50-58 | — |
| II | 55/60/65/70/75 | 55-63 | — |
| III | 60/65/70/75/80 | 60-68 | 融災黒潮の影響あり |

### 融災黒潮デバフ

| デバフ名 | 効果 |
|:---|:---|
| 狂気衰竭 | 狂気獲得量を低下 |
| 銀鍵振盪 | 銀鍵エネルギーを妨害 |
| 怒海狂瀾 | 深海界域特殊効果 |
| 猩紅哺育 | 血肉界域特殊効果 |
| 彼界終途 | 超維界域特殊効果 |
| 来自遺忘 | 混沌界域特殊効果 |

### 報酬しきい値

訓練値300以上で黒色貢物x2500 + 蔷薇金券x100,000（各段階）。III級300訓練値の黒色貢物は1500から2500に引き上げられました。

## 幻夢深潜（詳細）

Roguelikeモード。段位制（最高：幻夢伝説）。v1.5で大規模リニューアル。

### 主なメカニクス

| システム | 内容 |
|:---|:---|
| マップ | 戦争迷霧六角格マップ（毎日更新）。戦闘/精鋭/ボス/ショップ/休息/ランダムイベント/宝物ノード |
| 刻印 | 通常敵撃破後に獲得（3択）。優先：妙手 > 霊感 > 算力 > 鏡像パートナー > 回想 |
| 造物 | 精鋭敵撃破後に獲得（3択、通常2負面+1正面）。銀/金/呪いの3段階 |
| 黒印 | 戦闘ドロップ。融痕（交換所）で消費。優先：造物 > 覚醒 > 通常刻印 |

### 深潜ショップ

| アイテム | 価格 |
|:---|:---|
| 追念羽筆 | 融蝕之珠800=5本/週 |
| 碎裂余夢 | 融蝕之珠100/週 |
| 群星余夢 | 融蝕之珠25/週 |

## 攻略のポイント

1. **界域相性を確認** — 各ステージのボスには有利な界域が存在
2. **ギミック対策** — ステージ固有のギミックに対応した編成を組む
3. **周回効率** — スタミナ効率の良いステージを優先的に周回
4. **融災禁区は複数チーム必要** — 覚醒体を幅広く育成しておく

::: warning
チャレンジステージはメインステージをある程度進めないと解放されません。まずはメインステージを進めましょう。
:::
