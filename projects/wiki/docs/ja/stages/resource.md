# 資源ステージ

資源ステージは育成素材を効率的に収集するための常設ステージです。曜日によって入手できる素材が異なります。

## 曜日別ドロップ

| 曜日 | 入手可能素材 |
|:---:|:---|
| 月曜日 | *準備中...* |
| 火曜日 | *準備中...* |
| 水曜日 | *準備中...* |
| 木曜日 | *準備中...* |
| 金曜日 | *準備中...* |
| 土・日曜日 | 全素材 |

## 難易度

| 難易度 | 消費スタミナ | ドロップ量 |
|:---:|:---:|:---|
| 初級 | *準備中...* | 低レア素材中心 |
| 中級 | *準備中...* | 中レア素材含む |
| 上級 | *準備中...* | 高レア素材含む |

::: tip
土日は全素材が入手可能です。平日に足りない素材を土日にまとめて周回しましょう。
:::
