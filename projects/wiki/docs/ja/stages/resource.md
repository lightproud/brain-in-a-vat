# 資源ステージ

資源ステージは育成素材を効率的に収集するための常設ステージです。

## 融蝕之墟（Dissolution Ruins）

主要な素材獲得ステージ。墨諾芬（スタミナ）を消費して挑戦します。調査行動1-2通関で解放。

### ドロップ素材

| 素材カテゴリ | ドロップ内容 |
|:---|:---|
| 昇格素材 | 初階/中階/高階昇格匣子 |
| スキル素材 | 初階/中階/高階技能補給 |
| 界域素材 | 混沌残渣、深渕残渣、血腥残渣、宇宙残渣 |
| 教化素材 | 蒼白筆記、蒼白卷宗、蒼白典籍 |
| 通貨 | 蔷薇金券 |
| 経験素材 | 霊知制剤 I/II/III |
| 突破素材 | 各界域専用突破素材 |

### 界域別融蝕之墟

| ステージ | ドロップする界域素材 | 関連覚醒体 |
|:---|:---|:---|
| 混沌融蝕之墟 | 混沌残渣 | 混沌界域の覚醒体全般 |
| 深海融蝕之墟 | 深渕残渣 | 深海界域の覚醒体全般 |
| 血肉融蝕之墟 | 血腥残渣 | 血肉界域の覚醒体全般 |
| 超維融蝕之墟 | 宇宙残渣 | 超維界域の覚醒体全般 |

## 禁忌篆録（Forbidden Inscriptions）

通関後に繰り返し挑戦可能な素材ステージ。毎日リセット。

## 超験存在（Transcendent Existence）

| 項目 | 詳細 |
|:---|:---|
| タイプ | 週ボス |
| リセット | 毎週月曜日15:00 |
| 報酬 | 大量の育成素材 |

::: tip
超験存在は報酬が豊富です。毎週必ずクリアしましょう。
:::

## 素材収集の効率的なルート

| 優先度 | コンテンツ | 理由 |
|:---:|:---|:---|
| 1 | 超験存在（週ボス） | 報酬が非常に豊富。毎週必ずクリア |
| 2 | 融蝕之墟 | メイン素材源。必要な界域素材に合わせて周回 |
| 3 | 外勤代理（派遣） | 毎日精算で蔷薇金券・融蝕之珠・各種素材を獲得 |
| 4 | 課程 | 擬態凝晶・千面印章などコア素材を獲得 |

::: tip
土日は全素材が入手可能です。平日に足りない素材を土日にまとめて周回しましょう。素材は錬金システムで低級3個=高級1個に合成できます。
:::
