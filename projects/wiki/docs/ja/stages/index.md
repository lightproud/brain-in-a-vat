# ステージ一覧

忘却前夜（Morimens）のステージは大きく3種類に分かれます。

## ステージ分類

| ステージ | 説明 |
|:---|:---|
| [メインステージ](./main.md) | ストーリー進行に沿った通常ステージ |
| [資源ステージ](./resource.md) | 育成素材を効率的に収集 |
| [チャレンジステージ](./challenge.md) | 高難度ステージ。高レア素材がドロップ |
