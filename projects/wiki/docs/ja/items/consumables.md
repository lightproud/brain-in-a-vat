# 消耗品

戦闘や育成で使用する消耗品アイテムの一覧です。

## 回復アイテム

| アイテム名 | 効果 | 入手先 |
|:---|:---|:---|
| 小さな回復薬 | HP 500回復 | ショップ、メインステージ |
| 中回復薬 | HP 2,000回復 | ショップ、素材ステージ |
| 大回復薬 | HP 5,000回復 | チャレンジ報酬、イベント |
| 万能回復薬 | HP全回復 | イベント限定 |
| 蘇生の葉 | 戦闘不能から復活（HP50%） | ショップ、チャレンジ報酬 |

::: tip ヒント
回復アイテムは戦闘中のアイテム枠から使用できます。1戦闘につき使用回数制限があります（難易度により異なる）。
:::
