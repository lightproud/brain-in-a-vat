# 通貨一覧

忘却前夜のゲーム内通貨の一覧と入手方法です。

## 主要通貨

| 通貨 | 用途 | 主な入手先 |
|:---|:---|:---|
| 源液 (Silver Prime) | 課金通貨。召喚、通行証、外観、ギフトボックスの購入 | 課金購入 |
| 銀芯 (Silver Core) | 主要無課金通貨。流明之芯・無垢之芯への交換、墨諾芬回復（50銀芯=120墨諾芬） | デイリーミッション、メインステージ評価、手帳、幻夢深潜、イベント、通行証、交換コード |
| 墨諾芬 (Menophine) | スタミナ/行動力。幕間演習の消費に使用 | 自然回復、銀芯交換、アイテム/ギフトボックス |
| 蔷薇金券 (Rose Scrip) | 汎用通貨。ショップ購入、教化、錬金、素材合成、運命の輪重畳、密契写本（7500/回） | メインステージ初回通関、融蝕之墟、外勤代理、課程、運命の輪分解（青=1000, 紫=4000） |

## 召喚（ガチャ）通貨

| 通貨 | 用途 | 主な入手先 |
|:---|:---|:---|
| 流明之芯 (Lumen Core) | 限定池召喚通貨（1個=1回） | 銀芯交換（優先推奨）、イベント報酬、通行証 |
| 無垢之芯 (Pure Core) | 常驻池召喚通貨（1個=1回） | 銀芯交換、イベント報酬、キャラ初回昇格報酬、通行証無料トラック（40個）、チャプター初通関 |
| 沈澱幣 (Sediment Currency) | 沈澱ショップ交換。密契セット（猩红之拥）、追念羽筆（80沈澱=10本/週）、原初霊核購入 | 召喚（唤醒）時の副産物 |

## ショップ通貨

| 通貨 | 用途 | 主な入手先 |
|:---|:---|:---|
| 調査徽章 (Investigation Badge) | 調査ショップ。追念羽筆（800徽章=5本/週）の購入 | メインステージ三つ星通関、秘術統御、霊魂潜行三つ星、個人成就（300/個） |
| 無光幣 (Lightless Currency) | 無光ショップ。密契セット（远方的欢宴）、追念羽筆（800=5/週）の購入 | 無光秘境、界域異効区 |
| 融蝕之珠 (D-Jewels) | 深潜ショップ。追念羽筆（800=5/週）、碎裂余夢（100/週）、群星余夢（25/週）の購入。算出：(幻夢等級+2)x100 | 幻夢深潜（三級=500, 狂気=900）、外勤代理（毎日精算） |
| 相位幣 (Phase Coin) | 相位ショップ。密契セット（猩红之悸）、追念羽筆（800=5/週）の購入 | 相位対弈（PVP） |

::: tip
召喚通貨はメインステージ初回クリア報酬やデイリーミッションで入手できます。天井の30連分（＝召喚通貨30回分）を目標に貯めましょう。流明之芯は入手手段が限られるため、銀芯交換では流明之芯を優先してください。
:::
