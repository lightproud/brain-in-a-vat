# 造物（Creations）

**造物**は冶錬室で素材を合成して作成できる特殊アイテムです。

## 造物の用途

- **育成素材の変換** — 低レア素材を高レア素材に昇格
- **特殊効果アイテム** — 戦闘や調査行動で使用可能な消耗品
- **装備変換素材** — 密契・刻印の変換に使用

::: tip
造物のレシピは冶錬室の機能解放に伴って増えていきます。新しいレシピが追加されたら確認しましょう。
:::
