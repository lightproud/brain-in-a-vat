# 刻印一覧

刻印（Engravings）のセット一覧と推奨ビルドです。

## 刻印セット一覧

刻印は幻夢深潜（Roguelikeモード）内で獲得する一時的な強化効果です。探索中のみ有効で、永続的な装備ではありません。

データ収集中 -- 刻印の完全なデータベースは現在構築中です。以下は推奨サブステータスの参考情報です。

## 幻夢深潜での刻印優先度

| 優先度 | 刻印タイプ | 効果 |
|:---:|:---|:---|
| 1 | 妙手 | 高優先度の刻印効果 |
| 2 | 霊感 | カード関連の強化 |
| 3 | 算力 | 算力回復・追加 |
| 4 | 鏡像パートナー | パートナー召喚効果 |
| 5 | 回想 | その他の強化効果 |

## 役割別おすすめサブステータス

### 攻撃型

| 優先度 | サブステータス |
|:---:|:---|
| 1 | 会心率 |
| 2 | 会心ダメージ |
| 3 | 攻撃力% |
| 4 | 速度 |

### 防御型

| 優先度 | サブステータス |
|:---:|:---|
| 1 | HP% |
| 2 | 防御力% |
| 3 | 速度 |
| 4 | 攻撃力% |

### 支援型

| 優先度 | サブステータス |
|:---:|:---|
| 1 | 速度 |
| 2 | HP% |
| 3 | 攻撃力% |
| 4 | 会心率 |

### 副攻撃型

| 優先度 | サブステータス |
|:---:|:---|
| 1 | 速度 |
| 2 | 狂気蓄積率 |
| 3 | HP% |
| 4 | 攻撃力% |

::: warning
刻印の厳選は非常に時間がかかります。まずはメインステータスが正しいものを装備し、サブステ厳選は余裕がある時に行いましょう。
:::
