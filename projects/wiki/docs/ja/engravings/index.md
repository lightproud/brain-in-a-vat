# 刻印（Engravings）概要

**刻印**は覚醒体にサブステータスを付与する強化システムです。刻印の厳選は終盤コンテンツの攻略に不可欠な要素です。

## 基本情報

- 覚醒体に刻印を装備してサブステータスを付与
- 刻印にはメインステータスとサブステータスが存在
- サブステータスはランダムに決定される（厳選要素）

| 会心ダメージ | クリティカルダメージ倍率を上昇 |

## 厳選のポイント

::: tip
攻撃型覚醒体は「攻撃力%」「会心率」「会心ダメージ」のサブステを優先しましょう。防御型は「HP%」「防御力%」が重要です。

## 入手方法

- **チャレンジステージ** — 高レアリティ刻印がドロップ
- **冶錬室** — 不要な刻印を変換
- **イベント報酬** — 期間限定で入手可能
