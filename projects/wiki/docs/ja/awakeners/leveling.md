# 育成ガイド

覚醒体の育成方法を解説します。効率よくキャラクターを強化し、戦力を高めましょう。

## 育成要素一覧

| 育成要素 | 効果 | 上限 |
|:---|:---|:---:|
| レベルアップ | 基礎ステータス上昇 | プレイヤーLv依存 |
| 昇格（突破） | レベル上限解放・スキル解放 | 複数段階 |
| スキル強化 | カード性能向上 | Lv.6 |
| 啓霊 | 固有パッシブ解放 | 最大凸 |
| 運命の輪 | 専用装備でステータス強化 | — |
| 刻印 | サブステータス付与 | — |

## レベルアップ

経験値素材を消費して覚醒体のレベルを上げます。レベル上限はプレイヤーレベルに依存します。

::: tip
経験値素材は[資源ステージ](../stages/resource.md)や[派遣](../modes/dispatch.md)で効率的に収集できます。
:::

## 昇格（突破）

一定レベルに達すると昇格素材を消費して突破が可能になります。突破により：

- レベル上限が解放される
- 新しい指令カードが解放される
- ステータスボーナスが付与される

## スキル強化

指令カード・魂覚醒カードの性能を強化します。最大Lv.6まで強化可能です。

詳細は[カード強化](../cards/upgrade.md)をご覧ください。

## 啓霊（Enlightenment）

同一覚醒体の重複入手により啓霊レベルが上昇し、固有パッシブが解放されます。

詳細は[啓霊システム](./enlightenment.md)をご覧ください。

## 育成優先度

::: warning
JPサーバーはCNサーバーとバランス調整が異なる場合があります。最新のTier表を参考にしてください。
:::

### 序盤（Lv.1〜30）

1. メインアタッカー1体を集中育成
2. SSR覚醒体のレベルを優先
3. 指令カードのスキルLv.3を目指す

### 中盤（Lv.30〜60）

1. サブアタッカー・支援型の育成開始
2. 編成4体のレベルを均等に上げる
3. 運命の輪の装備を開始

### 終盤（Lv.60〜）

1. 啓霊による凸を進める
2. 刻印のサブステ厳選
3. 編成全体の最適化
