# 覚醒体図鑑

忘却前夜（Morimens）における**覚醒体**とは、プレイヤーが操作するキャラクターの総称です。各覚醒体は固有の界域・役割・カードセットを持ち、戦闘スタイルが大きく異なります。

## レアリティ

| レアリティ | 排出率 | 特徴 |
|:---:|:---:|:---|
| <span class="rarity-ssr">SSR</span> | 3% | 最高レアリティ。固有スキル・専用カードが強力 |
| <span class="rarity-sr">SR</span> | 15% | 高レアリティ。啓霊を重ねることで十分な戦力に |
| <span class="rarity-r">R</span> | 82% | 基本レアリティ。序盤の主力として活躍 |

## 界域（Realm）

覚醒体はそれぞれ4つの界域のいずれかに属します。

| 界域 | 英名 | テーマ | 特徴 |
|:---:|:---:|:---|:---|
| <span class="realm-badge realm-chaos">混沌</span> | Chaos | 狂気・混乱 | 狂気ゲージの蓄積が速く、バーストに特化 |
| <span class="realm-badge realm-aequor">深海</span> | Aequor | 深淵・海洋 | 防御・回復に優れたコントロール型 |
| <span class="realm-badge realm-caro">血肉</span> | Caro | 変容・肉体 | 高HP・自傷ギミックで爆発的火力 |
| <span class="realm-badge realm-ultra">超維</span> | Ultra | 次元・超越 | テクニカルな算力操作とカードドローに特化 |

## 役割（ロール）

- **攻撃型** — 高火力アタッカー。敵を速やかに撃破する
- **防御型** — タンク・シールド付与。味方を守る盾役
- **支援型** — バフ・デバフ・回復。編成の潤滑油
- **妨害型** — 状態異常・コントロール。敵の行動を制限

## 基本ステータス

| ステータス | 説明 |
|:---|:---|
| HP | 体力。0になると戦闘不能 |
| 攻撃力 | カードの基礎ダメージに影響 |
| 防御力 | 受けるダメージを軽減 |
| 速度 | 行動順序を決定 |
| 狂気耐性 | 狂気（Madness）の蓄積速度に影響 |

## カード構成

各覚醒体は以下のカードを所持します。

- **指令カード**（Command Card）— 通常攻撃・スキルに相当する基本カード
- **魂覚醒カード**（Spirit Awakening Card）— 狂気ゲージ100到達時に発動可能な必殺カード
