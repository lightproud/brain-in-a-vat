# 啓霊システム（Enlightenment）

**啓霊**は、同一の覚醒体を重複入手した際に発動する凸（重ね）システムです。啓霊レベルが上がるごとに固有のパッシブ能力が解放され、覚醒体の性能が大幅に向上します。

## 啓霊の仕組み

- 既に所持している覚醒体をガチャ等で再度入手すると、自動的に啓霊素材に変換される
- 啓霊素材を消費して啓霊レベルを上昇させる
- レアリティごとに最大啓霊レベルが異なる

## 啓霊レベルと効果

### SSR覚醒体

| 啓霊Lv | 必要数 | 解放効果 |
|:---:|:---:|:---|
| 1凸 | 2体目 | パッシブ1解放・ステータスボーナス |
| 2凸 | 3体目 | 指令カード強化・追加効果 |
| 3凸 | 4体目 | パッシブ2解放・大幅ステータス上昇 |
| 4凸 | 5体目 | 魂覚醒カード強化 |
| 5凸 | 6体目 | パッシブ3解放・最終形態 |

### SR覚醒体

| 啓霊Lv | 必要数 | 解放効果 |
|:---:|:---:|:---|
| 1凸 | 2体目 | パッシブ1解放 |
| 2凸 | 3体目 | 指令カード強化 |
| 3凸 | 4体目 | パッシブ2解放 |
| 4凸 | 5体目 | 魂覚醒カード強化 |
| 5凸 | 6体目 | パッシブ3解放 |

::: tip
SR覚醒体は排出率が高いため、啓霊を重ねやすくSSR覚醒体に匹敵する性能になることもあります。
:::

## 啓霊素材の入手方法

- **召喚（ガチャ）** — 重複入手時に自動変換
- **ショップ交換** — 各種通貨で啓霊素材を購入可能
- **イベント報酬** — 期間限定で配布されることがある

::: warning
SSR覚醒体の完凸（5凸）には同一キャラ6体分が必要です。無課金・微課金の場合は、汎用啓霊素材の活用を検討しましょう。
:::

## 啓霊の優先度

1. **メインアタッカーの2凸** — 指令カード強化で火力が大幅向上
2. **支援型の1凸** — パッシブ解放でサポート能力が拡張
3. **メインアタッカーの4凸** — 魂覚醒カード強化でバースト性能向上
