# コラボイベント

忘却前夜（Morimens）の他作品コラボレーション情報です。

## 沙耶の唄コラボ

クトゥルフ神話をテーマとした名作ビジュアルノベル「**沙耶の唄**」（ニトロプラス）とのコラボイベントです。

### コラボ概要

| 項目 | 詳細 |
|:---|:---|
| コラボ作品 | 沙耶の唄（Saya no Uta） |
| 開発元 | ニトロプラス（Nitroplus） |
| コラボ内容 | クトゥルフ神話という共通テーマを持つ《沙耶の唄》との跨界コラボレーション。コラボ限定の覚醒体が追加された |
| 入手方法 | コラボイベント期間中の限定ガチャ/イベント報酬 |

### コラボ限定コンテンツ

沙耶の唄とのコラボでは、以下の限定コンテンツが実装されました：

- **コラボ限定覚醒体** — 沙耶の唄から参戦したコラボキャラクター
- **限定ストーリーイベント** — コラボ専用のストーリーシナリオ
- **限定報酬** — コラボ期間中のみ入手可能なアイテム・外観

両作品はクトゥルフ神話（ラヴクラフト作品）をベースとした世界観を共有しており、忘却前夜の「融蝕」と沙耶の唄の「認知の変容」という共通テーマが交差するコラボレーションとなっています。

::: tip
コラボイベントは復刻の予定が不明な場合が多いです。開催期間中に限定覚醒体・報酬を確実に入手しましょう。
:::

::: warning
<span class="server-badge server-jp">JP</span> コラボイベントの開催時期はサーバーによって異なります。CNサーバーで先行開催されたコラボがJPサーバーに来るまでタイムラグがある場合があります。
:::
