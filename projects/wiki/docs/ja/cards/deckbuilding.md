# デッキ構築ガイド

忘却前夜のデッキ構築の基本とコツを解説します。ローグライト×デッキ構築の醍醐味を楽しみましょう。

## デッキ構築の基本

### 通常戦闘（ストーリー・ステージ）

通常の戦闘では、覚醒体の固有カードセットがそのままデッキとなります。カード強化とレベリングによりデッキ性能が向上します。

### 調査行動（Investigation）

ローグライトモードでは、ランの中でカードを取得・除去してデッキを構築します。

- 戦闘報酬やイベントでカードを追加
- 不要なカードはショップで除去可能
- ラン終了時にデッキはリセットされる

## 算力管理

毎ターンの算力は**5**です。デッキ構築では算力効率を常に意識しましょう。

| コスト帯 | 推奨枚数（割合） | 備考 |
|:---:|:---:|:---|
| 0コスト | 2〜3枚 | 算力不要だが効果は控えめ |
| 1〜2コスト | デッキの50〜60% | 主力。使いやすく回転率が高い |
| 3コスト | デッキの20〜30% | 中核カード。ターンの軸 |
| 4〜5コスト | 2〜4枚 | 強力だが算力を圧迫する |

::: tip
デッキが厚くなりすぎると主力カードのドロー率が下がります。調査行動では「カード追加」より「カード除去」の方が重要な場面も多いです。
:::

## 界域別デッキの特徴

### <span class="realm-badge realm-chaos">混沌</span>デッキ

- 狂気蓄積を加速するカードを軸に構築
- バーストの回転率を重視
- 低〜中コストカード中心で手数を増やす

### <span class="realm-badge realm-aequor">深海</span>デッキ

- シールド・回復カードで耐久しつつ戦う
- 敵のデバフを重ねて有利状況を作る
- 中コストカード中心のバランス型

### <span class="realm-badge realm-caro">血肉</span>デッキ

- 自傷カードで血肉固有バフを蓄積
- HPが低いほど火力が上がるギミックを活用
- ハイリスク・ハイリターンな構成

### <span class="realm-badge realm-ultra">超維</span>デッキ

- ドロー加速・算力回復カードで大量展開
- カードコンボを繋げるテクニカルなプレイ
- 高コストカードもドロー力でカバー

## デッキ構築のコツ

1. **軸を決める** — 勝ち筋を1つ明確にする
2. **薄いデッキが強い** — 不要カードは積極的に除去
3. **シナジーを意識する** — 相性の良いカード同士を採用
4. **算力カーブを意識する** — 毎ターン5算力を使い切れる構成に
5. **対策カードを忘れない** — 回復・防御カードも最低限は必要
