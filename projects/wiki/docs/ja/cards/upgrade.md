# カード強化

指令カードおよび魂覚醒カードは**最大Lv.6**まで強化可能です。強化により効果値が向上し、一部カードは追加効果が解放されます。

## 強化レベル

| レベル | 効果 | 必要素材 |
|:---:|:---|:---|
| Lv.1 | 基本性能 | — |
| Lv.2 | 効果値+10% | カード素材（低） |
| Lv.3 | 効果値+20% | カード素材（中） |
| Lv.4 | 効果値+35%、追加効果解放の場合あり | カード素材（高） |
| Lv.5 | 効果値+50% | カード素材（極） |
| Lv.6 | 効果値+70%、最終追加効果解放 | カード素材（極）＋特殊素材 |

::: tip
Lv.6への強化には通常のカード素材に加えて特殊素材が必要です。特殊素材は[チャレンジステージ](../stages/challenge.md)や[冶錬室](../modes/smelting.md)で入手可能です。
:::

## 強化素材の入手方法

| 素材 | 主な入手先 |
|:---|:---|
| カード素材（低） | メインステージ、資源ステージ |
| カード素材（中） | 資源ステージ、派遣 |
| カード素材（高） | チャレンジステージ、イベント |
| カード素材（極） | チャレンジステージ（高難度）、ショップ |
| 特殊素材 | チャレンジステージ、冶錬室 |

## 強化の優先度

1. **メインアタッカーの主力指令カードをLv.3に** — コスパが最も良い
2. **支援型のバフカードをLv.3に** — 編成全体の火力が向上
3. **メインアタッカーの魂覚醒カードをLv.4に** — バースト威力が大幅向上
4. **主力カードをLv.6に** — 終盤コンテンツの攻略に必要

::: warning
Lv.4以降は素材消費が急増します。複数キャラを均等に強化するより、主力1体を集中強化する方が効率的です。
:::
