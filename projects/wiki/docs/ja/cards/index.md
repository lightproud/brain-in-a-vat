# カードシステム概要

忘却前夜はローグライト×デッキ構築型のバトルシステムを採用しています。覚醒体が使用する**カード**を理解することが攻略の鍵です。

## カードの種類

### 指令カード（Command Card）

覚醒体の基本的な攻撃・スキルに相当するカードです。

- 各覚醒体は固有の指令カードセットを持つ
- 戦闘中にデッキからドローして手札に加え、算力を消費して使用
- 強化により最大**Lv.6**まで性能向上可能

### 魂覚醒カード（Spirit Awakening Card）

覚醒体の必殺技に相当する特別なカードです。

- **狂気ゲージが100**に到達すると使用可能
- 非常に強力な効果を持つが、使用後は狂気ゲージがリセットされる
- 啓霊4凸で強化版が解放される

## 算力（Arithmetica）

カードの使用にはコストとして**算力（Arithmetica）**を消費します。

- 毎ターン**5算力**が回復する
- 超維界域のシナジーや一部カード効果で追加回復が可能
- 高コストカードほど強力だが、算力管理が重要

## 狂気（Madness）ゲージ

- 戦闘中にカード使用・被ダメージ等で**狂気ゲージ**が蓄積
- **100に到達**すると魂覚醒カードが使用可能（バースト）
- 混沌界域の覚醒体は狂気蓄積が速い

## カード属性

各カードには以下の属性情報があります。

| 属性 | 説明 |
|:---|:---|
| コスト | 使用に必要な算力（0〜5） |
| 界域 | 所属する界域（混沌/深海/血肉/超維） |
| タイプ | 攻撃/防御/支援/特殊 |
| レアリティ | N/R/SR/SSR |
| 強化レベル | Lv.1〜Lv.6 |
