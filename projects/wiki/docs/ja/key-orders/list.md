# 鍵令一覧

銀鍵システムに関連する鍵令（Key Orders）の一覧です。銀鍵エネルギーが1000に達すると発動可能で、1ターンに1回使用できます（純混沌編成時は2回）。

## 汎用鍵令

| 鍵令名 | 界域親和 | 効果 | 入手方法 |
|:---|:---:|:---|:---|
| 鼠辈智慧 (Mouse Wisdom) | 汎用 | 算力を3点獲得 | メインステージ2-15通関 |
| 小小心愿 (Little Wish) | 汎用 | 覚醒体1名を選び、35点の狂気を付与 | メインストーリー |
| 酒馆之门 (The Tavern Door) | 汎用 | 「莉雅の硬貨」（算力2点獲得+2枚ドロー）を1枚捨札に追加 | 多維連動「逆流...」通関 |
| 注射防卫 (Injection Guard) | 汎用 | 防御/シールド関連効果（詳細未確認） | メインストーリー |
| 永世执念 (Eternal Obsession) | 汎用 | 臨時会心率25%+臨時力量7点獲得。ボス戦で当ターン未プレイなら死亡抵抗1回発動、成功時シールド138点獲得（1回限り） | メインストーリー |
| 不平等交换 (Unequal Exchange) | 汎用 | 正面効果（詳細未確認）。v1.1で「次ターンドロー-1」の副作用が撤廃 | メインストーリー |
| 仲夏之梦 (A Momentary Dream) | 汎用 | 1枚ドロー。指令カードまたは霊知覚醒なら算力消費0に変更、それ以外は捨てて効果を繰り返す。本戦闘中の界域精通を永続25点上昇 | メインストーリー |
| 最后的誓言 (The Final Oath) | 汎用 | 臨時力量23点+臨時会心率25%獲得。ボス戦で当ターン未プレイなら死亡抵抗1回発動、成功時バリア1層獲得（1回限り） | 拡張計画/ストーリー解放 |
| 回溯的终末 (Retrospective End) | 汎用 | 前ターンに最後にプレイした非派生指令カードの原本コピーを手札に追加（算力消費-1、消耗+虚無付き） | 拡張計画/ストーリー解放 |
| 不朽的葬仪 (Immortal Funeral Rites) | 汎用 | 全覚醒体に12点の狂気を付与。次ターン開始時に最大HP5%の献祭を受ける | 拡張計画/ストーリー解放 |
| 岁末花火 (Year-End Fireworks) | 汎用 | 味方1名の臨時強効+1、その味方の異なるスキル2枚を獲得。ターン終了時に全手札を捨てる | v1.1.11アップデート |
| 深宅往事 (Manor Memories) | 汎用 | 味方1名を選択、他の味方と全敵に「痴醉」1層付与（指令カード・狂気爆発のダメージ/回復/シールド効果-10%、駆散不可、上限2層） | ストーリー解放 |
| 泣血的圣心 (Bleeding Sacred Heart) | 汎用 | 算力消費が最も高い「スキル」を1枚ドローし、予備状態を付与 | ストーリー/イベント解放 |

| 短暂的永恒 (Brief Eternity) | <span class="realm-badge realm-ultra">超維</span> | 覚醒体1名を選び、その「打撃」と「防御」各2枚を手札に追加（消耗+虚無付き） | 超維界域：追加で臨時力量15点+臨時警戒4点獲得 | 拡張計画/ストーリー解放 |
| 灰雾面容 (Ashen Mist Visage) | <span class="realm-badge realm-ultra">超維</span> | 本戦闘中の会心率を永続5%上昇、「霊感」1枚を手札に追加 | 超維界域（超維ターン中）：追加で「霊感」1枚を手札に追加 | 拡張計画/ストーリー解放 |

| 玫瑰的胜利 (Victory of the Rose) | <span class="realm-badge realm-caro">血肉</span> | HP45点回復。HP25%以下の場合、回復量と同等の臨時力量を獲得 | 血肉界域：回復量+50%、直接回復せず猩紅溶鉱炉に蓄積 | 拡張計画/ストーリー解放 |
| 奥瑞塔的宝藏 (Aurita's Treasure) | <span class="realm-badge realm-aequor">深海</span> | 抽札堆から算力が最も低い3枚を手札に追加 | 深海界域：追加で触腕ダメージ+4~8点 | 拡張計画/ストーリー解放 |

## チーム別おすすめ鍵令

| チームタイプ | おすすめ鍵令 | 理由 |
|:---|:---|:---|
| 深海触腕チーム | 奥瑞塔的宝藏、蚀骨的拥抱 | 触腕生成と触腕ダメージ強化 |
| 血肉胚胎/融合チーム | 咆哮的血与沙、湖畔回眸、玫瑰的胜利 | 胚胎融合度・HP損失連動・猩紅溶鉱炉蓄積 |
| 超維カード回転チーム | 短暂的永恒、灰雾面容 | 追加手札生成・霊感カードで超維ターン支援 |
| 力量バーストチーム | 永世执念、最后的誓言 | 大幅な力量バフと会心率上昇 |
| 狂気速充チーム | 不朽的葬仪、咆哮的血与沙、小小心愿 | 狂気を素早く充填して狂気爆発を発動 |
| 汎用/初心者 | 仲夏之梦、鼠辈智慧、酒馆之门 | 算力補充とドローで全チームに対応 |

::: tip
鍵令の数値は守密人レベル60基準です。純混沌編成では毎ターン自動で銀鍵エネルギー250点が蓄積され、1ターンに2回鍵令を使用できます。
:::
