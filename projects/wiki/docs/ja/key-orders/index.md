# 鍵令（Key Orders）概要

**鍵令**は銀鍵（Silver Key）システムと連動する特殊な強化要素です。銀鍵ゲージの蓄積や消費に関わる効果を付与し、戦闘中の戦略幅を広げます。

## 銀鍵（Silver Key）システム

- 戦闘中に特定の行動で**銀鍵ゲージ**が蓄積される
- 銀鍵ゲージの最大値は**1000**
- ゲージ蓄積に応じて鍵令の効果が発動

## 鍵令の種類

| 分類 | 効果 |
|:---|:---|
| 蓄積系 | 銀鍵ゲージの蓄積速度を上昇 |
| 消費系 | 銀鍵ゲージを消費して強力な効果を発動 |
| 閾値系 | 銀鍵ゲージが一定値以上で常時効果を付与 |
