# 基地システム

プレイヤーの拠点「記憶の基地」の施設管理と生産について解説します。

## 概要

基地はプレイヤーレベル5で解放される拠点管理システムです。施設を建設・強化することで様々な資源を生産し、キャラクターとの交流も行えます。

## 施設一覧

### 資源施設

| 施設名 | 機能 | 解放条件 |
|:---|:---|:---|
| 忘却の鋳造所 | 忘却の欠片を時間経過で生産 | プレイヤーLv.5 |
| 経験の温室 | 経験の書を時間経過で生産 | プレイヤーLv.8 |
| 結晶の精錬所 | 属性結晶（小）を生産 | プレイヤーLv.12 |
| 武器工房 | 武器強化石を生産 | プレイヤーLv.15 |

### 機能施設

| 施設名 | 機能 | 解放条件 |
|:---|:---|:---|
| 交流広場 | キャラクターとの交流（好感度UP） | プレイヤーLv.5 |
| 厨房 | 贈り物（食べ物）の製作 | プレイヤーLv.10 |
| 図書館 | キャラストーリーの回想閲覧 | プレイヤーLv.10 |
| 訓練場 | 未編成キャラに経験値を自動付与 | プレイヤーLv.18 |
| ショップ | 基地通貨でアイテム購入 | プレイヤーLv.8 |
| 派遣所 | キャラクターを派遣して報酬獲得 | プレイヤーLv.20 |

### 装飾施設

| 施設名 | 機能 | 解放条件 |
|:---|:---|:---|
| 庭園 | 基地の外観をカスタマイズ | プレイヤーLv.10 |
| 居住区 | キャラクターの部屋を装飾 | プレイヤーLv.15 |

## 施設レベルアップ

各施設はレベルを上げることで生産効率や機能が向上します。

| 施設Lv | 必要素材 | 建設時間 |
|:---:|:---|:---:|
| Lv.2 | 忘却の欠片×1,000、基地資材×5 | 1時間 |
| Lv.3 | 忘却の欠片×3,000、基地資材×10 | 3時間 |
| Lv.4 | 忘却の欠片×8,000、基地資材×20 | 6時間 |
| Lv.5 | 忘却の欠片×15,000、基地資材×40 | 12時間 |
| Lv.6〜10 | *準備中...* | *準備中...* |

::: tip ヒント
施設レベルアップ中も資源の回収は可能です。建設時間はジュエルで短縮できますが、基本的には自然待機がおすすめです。
:::

## 派遣システム

キャラクターを派遣に出すと、一定時間後に素材や報酬を持ち帰ります。

| 派遣時間 | 報酬量 | 備考 |
|:---:|:---|:---|
| 1時間 | 少量 | 短時間で回収可能 |
| 4時間 | 中量 | バランス型 |
| 8時間 | 多量 | 就寝前に推奨 |
| 12時間 | 大量 | 最高効率 |

- 派遣にはキャラクターのレアリティが高いほど追加ボーナス
- 特定の属性と派遣先の組み合わせでボーナス報酬
- 派遣中のキャラクターは戦闘にも使用可能（他ゲームと異なる）

::: warning 注意
派遣報酬は回収しないと蓄積上限に達して生産が停止します。こまめにログインして回収しましょう。
:::
