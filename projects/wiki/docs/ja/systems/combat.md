# 戦闘システム

忘却前夜（Morimens）の戦闘システムについて解説します。

## 基本ルール

ターン制コマンドバトル方式です。各キャラクターは速度順に行動し、コマンドを選択して戦闘を進めます。

### 行動の種類

| コマンド | 説明 | SP消費 |
|:---|:---|:---:|
| 通常攻撃 | 基本攻撃。SPを回復する | 0（+10回復） |
| スキル1 | 戦闘スキル。様々な効果 | 20〜40 |
| スキル2 | 戦闘スキル。様々な効果 | 30〜50 |
| 奥義 | 必殺技。奥義ゲージ100%で発動 | 0（ゲージ消費） |
| 防御 | ダメージ50%軽減、SP+5 | 0 |

## SPシステム

- 各キャラクターの最大SP: 100
- 戦闘開始時のSP: 50
- 通常攻撃でSP +10回復
- 被弾時にSP +5回復
- 一部装備やパッシブでSP回復量が変動

## 奥義ゲージ

- 最大値: 100%
- ダメージを与える・受けるたびに蓄積
- キャラクターごとに蓄積速度が異なる
- 100%到達で奥義発動可能（任意のタイミングで使用可能）
- 奥義発動中は他のキャラクターの行動が中断されない

::: tip ヒント
奥義ゲージは通常攻撃よりスキルで攻撃した方が多く蓄積されます。SPと奥義ゲージのバランスを考えましょう。
:::

## 属性相性

属性相性によりダメージが増減します。

| 相性 | ダメージ倍率 |
|:---|:---:|
| 有利属性 | ×1.3（+30%） |
| 等倍 | ×1.0 |
| 不利属性 | ×0.7（-30%） |

詳細な属性相性表は[編成ガイド](../characters/team-building.md)を参照してください。

## ブレイクシステム

敵にはブレイクゲージが存在します。

1. 弱点属性で攻撃するとブレイクゲージが蓄積
2. ゲージが満タンになると「ブレイク状態」に移行
3. ブレイク中は防御力大幅ダウン＋1ターン行動不能
4. ブレイク中のダメージは **×1.5倍**

::: warning 注意
ボスのブレイクゲージは大量に蓄積が必要です。チーム全体で弱点属性を突く編成が有効です。
:::

## オート戦闘

- メニューからオートモードをON/OFF可能
- オート時はAIが最適な行動を自動選択
- 倍速機能: ×1、×2、×3から選択可能
- 掃討（スキップ）: ★3クリア済みステージで使用可能
