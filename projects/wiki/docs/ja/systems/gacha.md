# ガチャシステム

キャラクターと武器の入手方法であるガチャ（召喚）について解説します。

## ガチャの種類

| ガチャ名 | 対象 | 特徴 |
|:---|:---|:---|
| 恒常召喚「記憶の邂逅」 | キャラクター | 常時開催。限定キャラは含まない |
| ピックアップ召喚 | キャラクター | 期間限定。特定キャラの排出率UP |
| 武器召喚「追憶の鍛冶」 | 武器 | 常時開催。SSR武器を狙える |
| 限定武器召喚 | 武器 | 期間限定。専用武器の排出率UP |
| 初心者応援召喚 | キャラクター | アカウント作成後30日限定。割引あり |

## 天井（保証）システム

| 保証 | 条件 | 内容 |
|:---|:---|:---|
| 10連SR以上保証 | 10連召喚ごと | 最低1体はSR以上が確定 |
| SSR確定天井 | 90回召喚 | 90回以内にSSRが出なかった場合、90回目でSSR確定 |
| ピックアップ確定天井 | 180回召喚 | ピックアップ対象が確定で入手可能 |

::: tip ヒント
SSR天井は90回ですが、ピックアップ対象が出るとは限りません。確実にピックアップキャラを入手するには最大180回（天井2回分）のジュエルを用意しましょう。
:::

## 必要ジュエル数

| 召喚方法 | 消費ジュエル |
|:---|:---:|
| 単発召喚（1回） | 160 |
| 10連召喚 | 1,600（単発と同額） |
| 天井90回分 | 14,400 |
| 確定天井180回分 | 28,800 |

## ガチャチケット

| チケット種類 | 効果 |
|:---|:---|
| 通常ガチャチケット | 単発1回分（恒常ガチャのみ） |
| SR確定チケット | SR以上確定の単発1回 |
| SSR確定チケット | SSR確定の単発1回 |
| ピックアップチケット | ピックアップガチャで使用可能 |

## リセマラのおすすめ

::: tip リセマラ手順
1. ゲーム開始→チュートリアル完了（約10分）
2. プレゼントボックスからジュエル・チケットを受け取る
3. 初心者応援召喚で10連を1〜2回実行
4. SSRキャラが出なければアカウント削除→やり直し
:::
