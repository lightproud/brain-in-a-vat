# 好感度システム

キャラクターとの絆を深める好感度システムについて解説します。

## 概要

各キャラクターには好感度レベル（Lv.1〜Lv.10）があり、好感度を上げることでキャラストーリーの解放やステータスボーナスが得られます。

## 好感度の上げ方

| 方法 | 獲得好感度 | 備考 |
|:---|:---:|:---|
| パーティに編成して戦闘 | +5/戦闘 | デイリー上限あり（100/日） |
| 贈り物を渡す | +20〜100 | キャラの好みにより変動 |
| 好感度イベントをクリア | +50 | 特定条件で発生 |
| 基地で交流 | +10/回 | 1日3回まで |
| 誕生日プレゼント | +200 | 年1回、キャラの誕生日に |

## 好感度レベルと報酬

| レベル | 必要好感度 | 解放コンテンツ |
|:---:|:---:|:---|
| Lv.1 | 0 | 初期状態 |
| Lv.2 | 100 | キャラストーリー第1話、ステータス+1% |
| Lv.3 | 300 | ボイス追加、ステータス+2% |
| Lv.4 | 600 | キャラストーリー第2話、プロフィール情報追加 |
| Lv.5 | 1,000 | 専用称号解放、ステータス+3% |
| Lv.6 | 1,500 | キャラストーリー第3話 |
| Lv.7 | 2,200 | ボイス追加、ステータス+4% |
| Lv.8 | 3,000 | キャラストーリー第4話（SSR）/最終話（SR）、ボイス解放 |
| Lv.9 | 4,000 | 専用ホーム背景、ステータス+5% |
| Lv.10 | 5,500 | キャラストーリー最終話（SSR）、限定スキン、ステータス+6% |

## 贈り物システム

キャラクターごとに好みの贈り物カテゴリが設定されています。

### 贈り物カテゴリ

| カテゴリ | アイテム例 | 入手先 |
|:---|:---|:---|
| 花束 | 野の花束、輝く花束、永遠の花束 | ショップ、イベント |
| 書物 | 古い日記、魔導書、禁断の書 | ショップ、素材ステージ |
| 食べ物 | 手作りパン、高級スイーツ、伝説のフルコース | 基地の厨房で製作 |
| 宝飾品 | 銀のブローチ、宝石のペンダント、星の冠 | ショップ、チャレンジ報酬 |
| 想い出の品 | 写真、記憶のオルゴール、忘却の花 | イベント限定 |

### キャラクター別好みの贈り物

| キャラクター | 大好き（×3倍） | 好き（×2倍） | 普通（×1倍） |
|:---|:---|:---|:---|
| <span class="rarity-ssr">ルシェル</span> | 花束 | 想い出の品 | その他 |
| <span class="rarity-ssr">ノワール</span> | 書物 | 宝飾品 | その他 |
| <span class="rarity-ssr">エンバー</span> | 食べ物 | 宝飾品 | その他 |
| <span class="rarity-ssr">セレスティア</span> | 書物 | 花束 | その他 |

*その他のキャラクターの好みは準備中...*

::: tip ヒント
「大好き」カテゴリの贈り物を渡すと好感度が3倍になります。ショップで購入する前にキャラの好みを確認しましょう。
:::

## 好感度イベント

特定の好感度レベルに到達すると、基地で特別なイベントが発生します。

- イベントでは選択肢が表示される場合があります
- 正しい選択をすると追加の好感度ボーナスが得られます
- イベントは一度見ると「回想」メニューからいつでも再生可能

::: warning 注意
好感度イベントの選択肢による好感度変動は微量です。ストーリーの好みで選んでも大きな影響はありません。
:::
