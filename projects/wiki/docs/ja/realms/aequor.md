# 深海（Aequor）

<span class="realm-badge realm-aequor">深海</span>

**深海**はクトゥルフ神話における深淵の海洋生物や深きものどもをテーマとした界域です。防御・回復・コントロールに優れた持久戦特化の戦闘スタイルが特徴です。

## テーマ・世界観

深海界域はクトゥルフやダゴン、インスマスの住人たちに関連する深淵の力を宿す領域です。覚醒体たちは海洋の神秘的な力を操り、敵を圧倒します。

## 戦闘スタイル

- **シールド付与** — 味方にダメージを吸収するシールドを展開
- **回復能力** — HPを回復するカードが豊富
- **デバフ付与** — 敵の攻撃力・速度を低下させるコントロール
- **持久戦向き** — 長期戦で真価を発揮するが、瞬間火力は控えめ

## 界域シナジー

| 同界域数 | シナジー効果 |
|:---:|:---|
| 2体 | シールド量+15% |
| 3体 | シールド量+30%、回復効果+10% |
| 4体 | シールド量+50%、回復効果+25% |
