# 深海（Aequor）

<span class="realm-badge realm-aequor">深海</span>

**深海**はクトゥルフ神話における深淵の海洋生物や深きものどもをテーマとした界域です。触腕システムによる自動追撃と持久戦を得意とし、現環境（v2.4）で最強界域の一つです。

## テーマ・世界観

深海界域はクトゥルフやダゴン、インスマスの住人たちに関連する深淵の力を宿す領域です。覚醒体たちは海洋の神秘的な力を操り、敵を圧倒します。

## 戦闘スタイル

- **触腕システム** — 戦闘開始時に触腕1体を生成。深海覚醒体がカードをプレイすると追加で触腕を獲得。ターン終了時に前列の敵を自動攻撃。狂気爆発後にも追加攻撃発動
- **純深海共鳴** — 純深海編成時、深海精通属性の効果が2倍。戦闘開始時の自動生成触腕数も2倍
- **触腕のステータス計算** — 触腕の暴撃率・暴撃ダメージは出撃4体の覚醒体で均分。力量:触腕ダメージ=2:1、界域精通:狂気爆発後追加攻撃回数=2:1
