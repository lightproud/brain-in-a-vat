# 超維（Ultra）

<span class="realm-badge realm-ultra">超維</span>

**超維**はクトゥルフ神話における次元超越やヨグ＝ソトースの全知をテーマとした界域です。超維空間システムによる追加ターンを活用するテクニカルな戦闘スタイルが特徴です。

## テーマ・世界観

超維界域はヨグ＝ソトースに代表される時空を超越した存在の力を宿す領域です。覚醒体たちは次元の狭間から力を汲み上げ、通常の制約を超えたカードプレイを実現します。

## 戦闘スタイル

- **超維空間** — 超維界域才能発動後、チーム上部に「超維空間」が表示。毎ターン最初にプレイした指令カードの副本が蓄積され、上限到達で追加の超維ターンに突入。超維ターン中は蓄積した全カードが手札になる
- **純超維共鳴** — 純超維編成時、熟練度効果が2倍。追加ターン中の能力減少ペナルティが完全無効化
- **算力操作** — 算力を追加回復するカードが豊富
- **ドロー加速** — 追加ドローでデッキの回転率を上げる
- **テクニカル** — プレイング難度が高いが、上限火力も高い

## おすすめ編成

::: tip
超維界域は追加ターンを獲得できることが最大の強み。出力空間と緩衝時間を効果的に提供します。操作難度はやや高めですが、上限性能は十分に高いです。
:::

### 超維毒チーム（新規プレイヤー推奨）

| 枠 | キャラ | 役割 |
|:---:|:---|:---|
| 毒傷DPS（必須） | 旺达 (Wanda) | メインアタッカー |
| 蓄エネ補助（1覚醒） | 诺缔拉 (Nodera) | 蓄エネサポート |
| シールド（無料キャラ） | 艾瑞卡 (Erica) | 護盾 |
| 毒スタック/ツール | 宁菲亚 (Nymphia) / 温柯尔 (Winkle) | 毒付与・ツール |

旺达以外は代替可能。界域精通100以上推奨。宁菲亚+毒タロットでボスの無限遅延が可能。
