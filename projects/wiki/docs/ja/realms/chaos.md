# 混沌（Chaos）

<span class="realm-badge realm-chaos">混沌</span>

**混沌**はクトゥルフ神話における外なる神々の狂気をテーマとした界域です。銀鍵エネルギーの運用と反撃を中心に構築し、鍵令スキルで戦闘に参加します。

## テーマ・世界観

混沌界域はナイアルラトホテプやアザトースに代表される外なる神々の影響を色濃く受けた領域です。覚醒体たちは狂気に呑まれながらも、その力を武器として振るいます。

## 戦闘スタイル

- **死亡抵抗** — チームに混沌覚醒体がいれば追加で100%死亡抵抗（免死1回）を獲得
- **銀鍵共鳴**（純混沌編成）— 毎ターン自動で銀鍵エネルギー250点蓄積。1ターンに鍵令を2回使用可能（2回目は他の解放済み鍵令から選択）
- **高瞬間火力** — バースト時のダメージ倍率が高い
- **万能適応** — v2.0以降、他界域との共生効果を持つ
