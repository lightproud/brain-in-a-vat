# 血肉（Caro）

<span class="realm-badge realm-caro">血肉</span>

**血肉**はクトゥルフ神話における肉体変容やシュブ＝ニグラスの豊穣をテーマとした界域です。胚胎システムと吞噬メカニクスを駆使するハイリスク・ハイリターンの戦闘スタイルが特徴です。神父（萨尔瓦多）+専属運命の輪の実装後、深海と並ぶS級界域になりました。

## テーマ・世界観

血肉界域はシュブ＝ニグラスの眷属や、肉体の変容・進化に関わる存在の力を宿す領域です。覚醒体たちは自身の肉体を犠牲にしながら、その痛みを攻撃力へと変換します。

## 戦闘スタイル

- **胚胎システム** — 血肉界域才能発動後、チーム上部に胚胎ゲージが表示。血肉覚醒体のスキルで胚融合を蓄積し、100%到達で手札に「胚」カードを1枚追加
- **吞噬メカニクス** — 手札に「胚」カードがある時、血肉覚醒体の狂気爆発と組み合わせて「吞噬」効果を発動。胚を消費して超強力な効果を発動
- **胚胎回転加速** — 胚胎は全チームの狂気回復を加速し、大招（超限）の出力を強化
- **純血肉共鳴** — 純血肉編成時に特殊な強化効果を発動

## おすすめ編成

::: tip
血肉界域は上手難度が最も高い界域です。胚胎ゲージと血量のコントロールが鍵となります。界域精通34以上で毎ターン+35の胚融合値を自然蓄積できます。戦闘終了時には補助のエネルギー満タン+胚融合70-100%を維持するのが理想です。
:::
