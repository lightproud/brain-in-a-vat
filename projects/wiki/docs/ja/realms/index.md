# 界域概要

**界域**は忘却前夜の世界観とゲームシステムの根幹を成す4つの力の源泉です。すべての覚醒体・カード・装備はいずれかの界域に属し、界域ごとに異なる戦闘スタイルとシナジー効果を持ちます。

## 4つの界域

| 界域 | 英名 | テーマ | 戦闘スタイル |
|:---:|:---:|:---|:---|
| <span class="realm-badge realm-chaos">混沌</span> | Chaos | 狂気・混乱・外なる神 | 狂気バーストによる瞬間火力 |
| <span class="realm-badge realm-aequor">深海</span> | Aequor | 深淵・海洋・インスマス | 防御・回復・コントロール |
| <span class="realm-badge realm-caro">血肉</span> | Caro | 変容・肉体・シュブ＝ニグラス | 自傷ギミック・高HP戦略 |
| <span class="realm-badge realm-ultra">超維</span> | Ultra | 次元・超越・ヨグ＝ソトース | 算力操作・ドロー加速 |

## 界域の影響範囲

界域は以下のゲーム要素に関わります。

- **覚醒体** — 各覚醒体は1つの界域に属する
- **カード** — カードの効果は界域の特性を反映
- **界域シナジー** — 編成内の同界域覚醒体数でボーナスが発動
- **相性** — 一部コンテンツで界域間の有利不利が存在
- **装備（運命の輪・密契）** — 界域限定の装備が存在

## 界域シナジー早見表

| 同界域数 | 効果 |
|:---:|:---|
| 2体 | 基本シナジー発動 |
| 3体 | 強化シナジー発動 |
| 4体 | 最大シナジー発動（単色ボーナス） |
