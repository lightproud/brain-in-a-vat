# Tier表（覚醒体評価ランキング）

忘却前夜（Morimens）の覚醒体Tier表です。各コンテンツにおける評価をランキング形式で掲載しています。

::: warning
<span class="server-badge server-jp">JP</span> 本Tier表はJPサーバーの環境に基づいています。CN/グローバルサーバーとはバランス調整が異なるため、評価が異なる場合があります。
:::

## Tier基準

| Tier | 評価 |
|:---:|:---|
| **S+** | 環境最強。あらゆるコンテンツで最高のパフォーマンス |
| **S** | 非常に強力。大半のコンテンツで活躍 |
| **A** | 優秀。特定のコンテンツや編成で輝く |
| **B** | 平均的。育成すれば十分に活躍可能 |
| **C** | やや力不足。他に選択肢がない場合に |
