# Tier表（覚醒体評価ランキング）

忘却前夜（Morimens）の覚醒体Tier表です。各コンテンツにおける評価をランキング形式で掲載しています。

::: warning
<span class="server-badge server-jp">JP</span> 本Tier表はv2.4時点のデータに基づいています。CN/グローバルサーバーとはバランス調整が異なるため、評価が異なる場合があります。GameKee Wikiのランキングを参考にしていますが、最新版本と一致しない可能性があります。
:::

## Tier基準

| Tier | 評価 |
|:---:|:---|
| **T0** | 環境最強。あらゆるコンテンツで最高のパフォーマンス |
| **T1** | 非常に強力。大半のコンテンツで活躍 |
| **T2** | 優秀。特定のコンテンツや編成で輝く |
| **T3** | 平均的。育成すれば十分に活躍可能 |

## コンテンツ別Tier表

### 調査行動（Investigation）

主に24、图鲁、尤乌哈希といったT0メインCが活躍。莉兹の毒チームも低コストで攻略可能。凯刻斯の反撃チームは高練度で安定した通関が可能。

### 相位対局（Phase Chess / PvP）

PVPはBeta段階のため環境が流動的。御四家（拉蒙娜・萝坦・朵尔・奥吉尔）が基本編成。潘狄娅は嘲諷を無視して後衛を狙撃できるため高評価。

### 融災禁区（Dissolution Zone）

多チーム終局コンテンツ。覚醒体・運命の輪・鍵令の重複不可のため、幅広いキャラ育成が必要。24は単独でチームを支えられるため最優先。

### 幻夢深潜（Phantasmal Dive）

Roguelikeモード。深潜三杰（奥瑞塔・艾瑞卡・艾继丝）が高パフォーマンスを発揮。
