# 用語集

忘却前夜で使用される用語の中英日対照表。

## 角色

| 中国語 | English | 説明 |
|--------|---------|------|
| 唤醒体 | Awakener | 游戏中可操控角色的统称，游走在疯狂边缘的人形武器 |
| 守密人 | Keeper | 玩家扮演的角色身份 |

## 系统

| 中国語 | English | 説明 |
|--------|---------|------|
| 灵啡肽 | Menophine | 体力/行动力系统，有可调上限。主线不消耗 |
| 灵知深化 | Gnosis Enhancement | v1.1下半版本引入的系统，唤醒体等级上限提升至70级 |
| 界域 | Realm / Domain | 角色所属阵营类型：混沌、深海、血肉、超维。类似DBG中职业的概念 |
| 界域天赋 | Realm Talent | 运用相同界域的卡牌组合在一起时触发的职业效果 |

## 培养

| 中国語 | English | 説明 |
|--------|---------|------|
| 升格 | Ascension / Promotion | 突破角色等级上限的系统，首次在10级解锁 |
| 命轮 | Wheel of Destiny | 重复获取角色后可激活的额外能力，分R/SR/SSR品质 |
| 天赋 | Talents | v1.5引入的额外被动能力树，现有角色可免费获得 |
| 密契 | Sacred Covenant | 提供额外角色增强与搭配思路的系统 |

## 战斗

| 中国語 | English | 説明 |
|--------|---------|------|
| 刻印 | Engravings | 普通敌人击败后对卡牌附加额外效果 |
| 吞噬 | Devour | 血肉界域核心：消耗「胚」卡配合狂气爆发触发超强力效果 |
| 应急灵知体 | Emergency Spirit Body | HP归零时消耗1个复活，恢复全状态，每场战斗最多5个 |
| 灵之觉醒牌 | Spirit Awakening Card | 每个唤醒体的特殊能力卡牌，区别于4张基础指令牌 |
| 狂气 | Madness | 角色通过出牌积累的能量，满后可释放狂气爆发 |
| 狂气爆发 | Madness Burst | 角色大招，狂气满后发动的强力技能 |
| 算力 | Computation | 使用卡牌时消耗的核心资源 |
| 胚 | Embryo | 血肉界域核心机制，胚融合达100%时获得「胚」卡片 |
| 触腕 | Tentacle | 深海界域核心机制，回合结束时自动攻击前排敌人 |
| 超维空间 | Hyperdimension Space | 超维界域核心机制，蓄积卡牌副本后进入额外超维回合 |
| 超限因果 | Transcendence Causality | v1.5与超限爆发同时引入的新系统 |
| 超限爆发 | Reproduction Frenzy | v1.5引入的新终极技能系统，部分角色可解锁 |
| 造物 | Artifact / Relic | 精英敌人击败后获得的特殊效果物品，分正面和负面 |
| 钥令 | Key Order | 守密人的独属能力，消耗银钥能量在战斗中发动 |
| 银钥共鸣 | Silver Key Resonance | 纯混沌编成时的特殊效果，每回合自动积攒250点银钥能量 |
| 银钥能量 | Silver Key Energy | 守密人的专属能量，用于发动钥令技能 |
