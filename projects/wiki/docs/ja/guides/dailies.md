# デイリー攻略

毎日効率的にプレイするためのデイリールーティンガイドです。

## デイリーチェックリスト

### 必須（毎日必ず実行）

- [ ] デイリーミッションの完了
- [ ] スタミナの消費（資源ステージ or 調査行動）
- [ ] 派遣の回収・再設定
- [ ] ショップの無料枠確認
- [ ] フレンドへの挨拶（フレンドポイント）

### 推奨（時間がある時に）

- [ ] 調査行動（Investigation）のデイリー分消化
- [ ] 相位対局（PvP）のデイリー対戦
- [ ] イベントステージの消化
- [ ] 意識潜游の進行

### 週間タスク

- [ ] ウィークリーミッションの完了
- [ ] 資源ステージの曜日素材周回
- [ ] チャレンジステージの週間報酬

## スタミナ効率

| 用途 | 優先度 | 備考 |
|:---|:---:|:---|
| イベントステージ | 最高 | 期間限定のため最優先 |
| 資源ステージ（不足素材） | 高 | 育成に必要な素材を優先周回 |
| チャレンジステージ | 高 | 密契・刻印の厳選 |
| メインステージ | 中 | 未クリア分がある場合 |

::: tip
スタミナは溢れさせないことが最も重要です。就寝前にスタミナを消費し、長時間派遣をセットしましょう。
:::

## 効率的なプレイ時間

| 時間帯 | やること | 所要時間 |
|:---|:---|:---:|
| 朝 | 派遣回収、デイリーミッション確認 | 5分 |
| 昼 | 短時間派遣回収、スタミナ消費 | 10分 |
| 夜 | メインプレイ（調査行動・PvP等） | 30〜60分 |
| 就寝前 | スタミナ消費、長時間派遣設定 | 5分 |
