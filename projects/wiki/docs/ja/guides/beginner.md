## リセマラ

### リセマラの目安

- **SSR覚醒体1体**を目標にリセマラ
- SSR排出率は**3%**、30連で天井
- 初回ガチャで引き直し機能がある場合はそちらを活用

## 序盤の進め方

### 1日目：チュートリアル〜メインステージ序盤

1. チュートリアルを完了する
2. 初回召喚（ガチャ）を引く
3. メインステージを進められるところまで進める
4. 各種コンテンツの解放を目指す

### 2〜3日目：育成開始

1. メインアタッカーのレベルを優先的に上げる
2. 指令カードをLv.3まで強化
3. 資源ステージの周回を開始
4. 派遣を設定する

### 1週間目：コンテンツ本格参加

1. 調査行動（ローグライト）に挑戦
2. 密契の収集・装備を開始
3. 刻印の基本装備を整える
4. 編成を4体揃える

## 育成の優先順位

1. **メインアタッカー** — レベル・カード強化を最優先
2. **ヒーラー/シールダー** — 2番手で育成
3. **サブアタッカー** — 3番手
4. **妨害型** — 最後でOK

## 算力（Arithmetica）の基本

- 毎ターン**5算力**が回復
- カードのコスト分だけ消費
- 5算力を使い切るのが基本。余らせるのはもったいない

## 狂気（Madness）の基本

- **100まで蓄積**すると魂覚醒カード（バースト）が使用可能
- 混沌界域の覚醒体は蓄積が速い
- バーストは非常に強力。意図的に溜める戦術も有効

## 召喚（ガチャ）の心得

- SSR確率**3%**、**30連で天井**
- 天井分の通貨を貯めてから引く
- ピックアップバナーを優先

::: tip
初心者は恒常ガチャを引かず、ピックアップバナーに通貨を集中投下するのがおすすめです。
:::

## やってはいけないこと

- 素材を複数キャラに分散投資（主力1体に集中すべき）
- 天井に届かない状態でガチャを引く
- R覚醒体への過度な投資（序盤の繋ぎとして使い、早めにSR/SSRに切り替える）
- 刻印の早期厳選（終盤コンテンツ。序盤はメインステ正しいだけで十分）
