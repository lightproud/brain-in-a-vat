# 冶錬室（Smelting）

**冶錬室**は不要な素材や装備を変換・合成するシステムです。余った素材を有効活用し、必要なアイテムを入手できます。

## 基本機能

| 機能 | 説明 |
|:---|:---|
| 素材変換 | 低レアリティ素材を高レアリティ素材に変換 |
| 装備変換 | 不要な密契・刻印を別のものに変換 |
| 合成 | 複数の素材を合成して特殊素材を作成 |
| 分解 | 不要な装備を素材に分解 |

## 冶錬室の活用

::: tip
刻印の厳選で大量に出る不要品は冶錬室で変換しましょう。完全な無駄にはなりません。
:::

1. **不要な密契の変換** — サブステが悪い密契を変換して再挑戦
2. **低レア素材の昇格** — 余った低レア素材を高レアに変換
3. **特殊素材の合成** — カードLv.6強化に必要な特殊素材を作成
