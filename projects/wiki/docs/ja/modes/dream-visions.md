# 異夢視界（Dream Visions）

**異夢視界**は期間限定で開催される特殊ステージモードです。通常とは異なるルールや制限のもとで戦闘を行い、限定報酬を獲得できます。

## 基本情報

- 期間限定で開催される特殊コンテンツ
- イベントごとに異なるルール・ギミックが設定される
- 限定報酬（覚醒体・密契・素材等）を入手可能
