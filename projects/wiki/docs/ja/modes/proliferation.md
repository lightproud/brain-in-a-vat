# 繁殖狂乱（Proliferation）

**繁殖狂乱**はウェーブ制のサバイバルモードです。次々と押し寄せる敵の群れを撃退し、生き残った波数に応じて報酬を獲得します。

## 基本ルール

- ウェーブ（波）制で敵が次々と出現
- ウェーブが進むごとに敵の数・強さが増加
- 全滅するか撤退するまで継続
- 到達ウェーブに応じて報酬を獲得

## ウェーブ構成

| ウェーブ | 特徴 |
|:---:|:---|
| 1〜5 | 通常の雑魚敵。ウォームアップ |
| 6〜10 | エリート敵が混在。編成力が試される |
| 11〜15 | ミニボスが出現。高難度 |
| 16〜20 | 大量の敵+ボス級。最高難度 |
| 21〜 | エンドレス。ランキング用 |

## 攻略のコツ

1. **範囲攻撃が有効** — 大量の敵を一掃できるカードを優先
2. **持久力が重要** — 回復・シールド手段を確保
3. **狂気バーストの温存** — ミニボスウェーブに合わせてバーストを温存

::: tip
繁殖狂乱は血肉界域の自傷ギミックと相性が良いです。高HPを活かした長期戦が有効です。
:::
