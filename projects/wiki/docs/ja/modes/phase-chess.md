# 相位対局（Phase Chess）— PvP

**相位対局**は忘却前夜のPvP（対人戦）モードです。他のプレイヤーとリアルタイムまたは非同期で対戦し、ランキングを競います。

## 基本ルール

- プレイヤー同士が覚醒体編成（最大4体）で対戦
- 通常の戦闘システムをベースにPvP向け調整が入っている
- シーズン制でランキングが更新される

## 対戦形式

### リアルタイム対戦

- マッチングにより対戦相手が決定
- ターンごとに制限時間あり
- 高ランク帯で解放

### 非同期対戦

- 相手のAI操作編成と対戦
- 時間制限なしでじっくりプレイ可能
- 低ランク帯から利用可能

## PvPのコツ

1. **相手の界域を確認** — 編成を見て対策を立てる
2. **速度が重要** — 先手を取ることで有利な展開を作れる
3. **妨害型を活用** — PvPではコントロール能力が特に重要
4. **算力管理を徹底** — 限られた算力で最大効率のプレイを
