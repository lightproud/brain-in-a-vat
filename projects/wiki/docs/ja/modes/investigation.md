# 調査行動（Investigation）

**調査行動**は忘却前夜のメインコンテンツであるローグライトモードです。各ランでデッキを構築しながらマップを探索し、最奥のボスを撃破することが目的です。

## 基本ルール

- ランごとに初期デッキからスタート
- 道中で**カードの追加・除去・強化**を行いデッキを構築
- マップはランダム生成。戦闘・イベント・ショップ等のノードが配置される
- ラン終了時（クリアまたは敗北）にデッキはリセット

## マップノード

| ノード | 説明 |
|:---|:---|
| 戦闘 | 敵との戦闘。勝利でカード・素材報酬 |
| エリートバトル | 強敵との戦闘。高報酬 |
| ボス | マップ最奥の強敵。撃破でクリア |
| イベント | ランダムイベント。選択肢により結果が変化 |
| ショップ | カード購入・除去・強化が可能 |
| 休息 | HP回復またはカード強化を選択 |
| 宝箱 | ランダム報酬を入手 |

## デッキ構築のコツ

1. **序盤は攻撃カードを優先** — 戦闘を速く終わらせて被ダメージを抑える
2. **不要なカードは除去** — デッキが厚いと主力カードが引けない
3. **シナジーを意識** — カード同士の相性を考えて取得する
4. **ボス戦を見据える** — 道中だけでなくボス攻略を意識した構築を

::: tip
調査行動のドロップ報酬は密契や育成素材の主要な入手源です。毎日のデイリー消化で必ずプレイしましょう。
:::
