# 召喚システム（ガチャ）

忘却前夜の覚醒体・運命の輪の入手方法である召喚（ガチャ）システムを解説します。

## 基本確率

| レアリティ | 排出率 |
|:---:|:---:|
| <span class="rarity-ssr">SSR</span> | **3%** |
| <span class="rarity-sr">SR</span> | 15% |
| <span class="rarity-r">R</span> | 82% |

::: tip
v2.0でSSR基礎排出率が1.02%から3%に大幅向上しました。
:::

## 天井（保証）システム

- **30連（30回召喚）で天井**が発動（v2.0改定、旧v1.xでは80連）
- 天井到達時、ピックアップSSR覚醒体が確定入手
- 天井カウントはバナーごとに独立（限定池と常驻池は別計算）

### すり抜け救済仕様

| 回数 | ピックアップ確率 |
|:---:|:---|
| 1回目のSSR | 33.33%がピックアップ対象 |
| 2回目のSSR（1回目すり抜け時） | 66.66%がピックアップ対象 |
| 3回目のSSR（2回目もすり抜け時） | **100%**ピックアップ確定 |

最大120連（4回の30連サイクル）以内にピックアップキャラが確定入手できます。また、同じピックアップ対象が2回連続で出た場合、次回は別のピックアップ対象が確定します。

## バナーの種類

| バナー | 通貨 | 特徴 |
|:---|:---:|:---|
| 灵识黒潮（常驻池） | 無垢之芯 | 常驻SSRキャラ池。UP機制なし。30連保底SSR。5回唤醒でSSR+運命の輪保底 |
| 限定活動唤醒（限定池） | 流明之芯 | 新限定キャラ初登場池。限定SSRは常驻に入らない |
| 万華追憶（限定専属池） | 流明之芯 | 純限定キャラ池。常驻SSRを含まない。30連以内に限定SSRキャラまたは運命の輪を保底 |
| 復刻池 | 流明之芯 | 過去の限定キャラ復刻 |
| 必得唤醒（課金保底池） | 源液x980/回 | 5回で限定SSRキャラ2体+限定SSR運命の輪3本を保底。旧世篇/星辰篇・男/女子池に分類 |
| 自選唤醒 | 各種 | イベント限定。SSRキャラ1体+専属運命の輪をUP対象として自選可能 |

## 召喚通貨

| 通貨 | 用途 | 入手方法 |
|:---|:---|:---|
| 源液 (Silver Prime) | 課金通貨。必得唤醒等に使用 | 課金購入 |
| 銀芯 (Silver Core) | 主要無課金通貨。流明之芯・無垢之芯に交換（約200銀芯=1回分） | デイリーミッション、メインステージ評価（100/関）、手帳、幻夢深潜、イベント、通行証、交換コード |
| 流明之芯 (Lumen Core) | 限定池召喚通貨（1個=1回）。入手手段が限られるため優先交換推奨 | 銀芯交換、イベント報酬、通行証 |
| 無垢之芯 (Pure Core) | 常驻池召喚通貨（1個=1回） | 銀芯交換、イベント報酬、キャラ初回昇格、通行証無料トラック（40個）、チャプター初通関 |
| 沈澱幣 (Sediment Currency) | 召喚副産物。沈澱ショップで素材交換 | 唤醒時の副産物 |

## 召喚の心得

::: warning
<span class="server-badge server-jp">JP</span> JPサーバーの実装順序はCNサーバーと異なる場合があります。今後実装予定のキャラクターを見越して通貨を温存するのも戦略の一つです。
:::

1. **天井分の通貨を確保してから引く** — 30連分を貯めてからガチャに臨む
2. **ピックアップを狙う** — 恒常ガチャより期間限定ピックアップの方が効率的
3. **運命の輪は余裕がある時に** — キャラクター優先。装備は後から入手手段がある
4. **啓霊は計画的に** — 完凸を狙うか、新キャラを広く集めるか方針を決める
5. **常驻池はキャラ池優先** — 無垢之芯でまずキャラ池を回し、核心キャラを確保
6. **F2Pプレイヤーは24を最優先** — 次いで墨菲を確保
