# 運命の輪（Wheels of Destiny）概要

**運命の輪**は覚醒体専用の装備システムです。各覚醒体に対応した運命の輪を装備することで、ステータスの強化や特殊効果を得られます。

## 基本情報

- 各覚醒体には**専用の運命の輪**が存在する
- 運命の輪はガチャ（召喚）やイベントで入手可能
- 強化・突破により性能が向上する

## レアリティ

| レアリティ | 特徴 |
|:---:|:---|
| <span class="rarity-ssr">SSR</span> | 専用パッシブ効果あり。最高性能 |
| <span class="rarity-sr">SR</span> | 汎用性が高く実用的 |
| <span class="rarity-r">R</span> | 基本的な性能。繋ぎとして使用 |

## 運命の輪の効果

- **基礎ステータス上昇** — HP・攻撃力・防御力等のステータスボーナス
- **専用パッシブ** — SSR運命の輪のみ。覚醒体固有の特殊効果を付与
- **界域ボーナス** — 同界域の運命の輪を装備することで追加ボーナス
