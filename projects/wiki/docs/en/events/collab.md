# Collaboration Events

Morimens has featured cross-media collaboration events with other intellectual properties. These events typically include exclusive Awakeners, cards, story content, and cosmetics themed around the collaborating work.

## Collaboration History

### Saya no Uta (沙耶之歌)

Morimens has collaborated with **Saya no Uta** (Saya's Song), a visual novel by Nitroplus written by Gen Urobuchi. Given Morimens' Lovecraftian themes and Saya no Uta's cosmic horror narrative, this collaboration is a natural thematic fit.
