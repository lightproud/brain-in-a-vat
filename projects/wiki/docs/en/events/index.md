# Events

Morimens regularly runs limited-time events that offer exclusive rewards, story content, and gameplay challenges.

## Quick Navigation

| Page | Description |
|------|-------------|
| [Current Events](./current.md) | Events currently in progress |
| [Event History](./history.md) | Archive of past events |
| [Collaboration Events](./collab.md) | Cross-media collaboration events |

## Event Types

| Type | Description |
|------|-------------|
| Story Events | Limited-time narrative content with unique stages |
| Challenge Events | Special difficulty encounters with ranking rewards |
| Login Events | Log in daily during the event period for free rewards |
| Collaboration Events | Cross-media collaborations with other IPs |
| Celebration Events | Anniversary, milestone, and holiday events |

::: tip
Events are the best source of free premium currency, summon tickets, and exclusive items. Always participate in active events, even if you cannot clear the hardest difficulties.
:::
