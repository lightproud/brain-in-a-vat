# Event History

This page archives information about past events in Morimens for reference. Events are listed by version in chronological order.

## Version Timeline

### v2.5 (2026, Preview)

- New character Arachne (Ultra) previewed

### v2.4 -- Stellar Chapter 4 "Paradise" / 2nd Anniversary (2025-2026)

- Stellar Chapter 4 "Paradise" main story (4-0 to 4-16) -- largest difficulty spike chapter
- 2nd Anniversary celebration event
- New character type **Chorus** introduced
- New limited characters: Murphy: Fauxborn (Aequor, Chorus), Xu (Caro, Chorus)
- Divine Realm system: Chorus characters can activate enhanced realm mechanics
- 2nd Anniversary event "By the Silver Key": use Key Orders 200 times to self-select 1 past limited character
- JP server announcement: August 19, 2025 launch by AltPlus Inc.

### v2.3 -- Stellar Chapter 3 (2025)

- Stellar Chapter 3 main story (3-1 to 3-11)
- New limited characters: Doresain (Caro), Mouchette (Caro), Pollux (Ultra, v2.3.4), Pickman (Aequor, v2.3.2)
- Multiple character rerun banners

### v2.2 -- Stellar Chapter 2 "Metamorphosis" (2025)

- Stellar Chapter 2 main story (2-1 to 2-18)
- New limited characters: Kathigu-Ra (Chaos), Castor (Ultra, v2.2.2)
- Stellar chapter new wheels

### v2.1 -- Stellar Chapter 1 (2025)

- Stellar Chapter 1 main story (1-1 to 1-14)
- New limited characters: Clementine (Ultra), Corposant (Aequor)
- Stellar chapter new wheels

### v2.0 -- Stellar Ascension (2025-2026)

- New main story arc "Stellar Chapter" launched
- **Major system overhaul:**
  - Team composition rules changed: any 2 different realms can now form a team
  - Key Order system introduced: Keeper's exclusive combat abilities
  - Chaos realm reworked: Silver Key Resonance mechanic
  - SSR base rate increased from 1% to 3%, pity reduced from 80 to 30 pulls
  - Most characters upgraded to SSR rarity
  - R-quality wheels completely reworked with exploration-changing effects
  - Extra wheel slot unlocked at certain progression
  - Growth system streamlined
- New limited character: Doll: Inferno (Chaos)

### v1.5 (2025)

- New Awakeners: Horla (Ultra), Karen (Chaos, also known as Ryker)
- New systems: "Overload Burst" and "Overload Causality"
- Existing characters receive free new talents
- Ultra realm enhanced
- Burden reduction plan implemented
- Wanda and Creation system adjustments
- Phantasmal Dive overhaul
- Multiple rerun banners: Tulu, Daffodil, Sorel, Lily, Murphy

### v1.4 (2024-2025)

- Miryam enhanced: "Supreme Life Sacrifice" skill adjusted to piercing damage; Gnosis Awakening and talents enhanced
- v1.4.4 update with ongoing balance changes
- Half-anniversary rewards: 5,000 Silver login bonus, 1 past limited self-select, SSR wheel "Polar Night and Dawn" gift

### v1.3 (2024)

- Ongoing content and system updates
- Multiple maintenance patches (v1.3.0, v1.3.1, v1.3.2, v1.3.11)

### v1.2 (2024)

- Dissolution Zone unlock requirement lowered (5-4 to 4-3)
- Phantasmal Dive all Creation sets launched with balance adjustments

### v1.1 (Early 2024)

- New characters: Alva, Nodera
- First limited-time event "Magic of Stories" (v1.1 second half)
- **Dissolution Zone** (endgame multi-team challenge) launched
- **Gnosis Deepening** system (level cap raised to 70)
- New character: Lily (Chaos, Defense, via "Untainted Blossom" event)
- Daffodil character rework
- Extensive gameplay adjustments
- Dense online maintenance updates (v1.1.9 through v1.1.17)

### v1.0 -- Official Launch (2023)

- Game officially launched
- Oblivion Chapter (Prologue through Chapter 5) main story
- Initial character roster: Ogier, Doll, Ramona, Ramona: Timeworn, Lotan, and others
- Four realm combat system
- Phase Chess PVP mode
