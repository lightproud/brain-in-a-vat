# Covenants

**Covenants** (密契) are universal equipment in Morimens. Unlike Wheels of Destiny, Covenants can be equipped by any Awakener, making them flexible gear options for your entire roster.

## Overview

| Aspect | Detail |
|--------|--------|
| Type | Universal equipment |
| Acquisition | Stages, events, crafting |
| Equip Limit | Varies by slot |
| Tradeable | No |

## How Covenants Work

- Covenants provide **stat bonuses** and **passive effects** that are not tied to any specific Awakener.
- Any Awakener can equip any Covenant, allowing you to redistribute gear freely.
- Covenants are ideal for filling equipment slots when you lack a character-specific Wheel of Destiny.

## Obtaining Covenants

| Source | Notes |
|--------|-------|
| Stage Drops | Farm from various stage types |
| Event Rewards | Limited-time event shops and milestones |
| [Smelting Room](../modes/smelting.md) | Craft Covenants using gathered materials |
| Investigation | Reward from [Investigation](../modes/investigation.md) runs |

## Covenants vs. Wheels of Destiny

| Feature | Covenants | Wheels of Destiny |
|---------|-----------|------------------|
| Equip restriction | None (universal) | Character-specific |
| Source | Stages, events, crafting | Primarily gacha |
| Power level | General purpose | Higher (tailored passives) |
| Flexibility | Can swap between characters | Locked to one Awakener |

::: tip
Covenants are the backbone of equipment for free-to-play players. Since they can be moved between Awakeners, a single strong Covenant set can gear up whichever character you need for the current content.
:::
