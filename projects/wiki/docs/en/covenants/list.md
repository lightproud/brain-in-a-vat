# Covenants List

A complete list of all 17 Covenant sets in Morimens. Covenants unlock at account level 35. Each set has 6 pieces, providing 3-piece and 6-piece set bonuses.

## Covenant Sets

| Covenant Name | Role | Effect | Source |
|---------------|------|--------|--------|
| Scarlet Palpitation | DPS (Main Carry) | Provides dual crit (Crit Rate + Crit DMG). Universal set for all main carries -- suits 24, Lotan, Uvhash, etc. | Shop Exchange / Investigation Exchange / Phase Exchange |
| Scarlet Embrace | Support / Sub-DPS | Suited for support characters who actively play cards. Effect is wasted if the wearer does not play cards. | Sediment Exchange |
| Twisted Twins Black | DPS / Support | Copies the wearer's Strike card and adds it to the deck. Suits Lotan, Aigis. | Stage Drop |
| Twisted Twins White | Tank / Defense | Copies the wearer's Defense card and adds it to the deck. Suits Ogier, Nodera. | Stage Drop |
| Steppe Wolf | DPS / Poison | DMG Amplification + counter/poison bonus. Suits Liz, Wanda. v1.4.4: 6-piece no longer requires specific attribute. | Rose Scrip Shop |
| Deus Ex Machina | Support | AP/Arithmetica related effects. Support character exclusive. | Stage Drop |
| Returnal Line | Deep Sea DPS | Power bonus or additional tentacle attack count. Deep Sea character exclusive. | Stage Drop |
| Burial Ground Whispers | Survival / Utility | 6-piece: +25% Death Resist at stage start; gain 50 Aliemus after triggering Death Resist. | Stage Drop |
| Curse Rabbit | Healer | Heal + shield bonus. Suits Salvador, Doll, Celeste. v1.4.4: 6-piece no longer requires specific attribute. | Deep Dive Shop |
| Distant Feast | Tank / Defense | Large defense bonuses. Nodera post-awakening can provide 200+ shield per defense card. v1.4.4: 6-piece no longer requires specific attribute. | Lightless Shop |
| Unstained Chronicle | Support | Support-type character exclusive set. | — |

| Antinomy | DPS | 6-piece (team-unique): Aliemus cap and Overload cost +20; command card crit rate, crit DMG, shield, heal +20%. | — |
| Dream of Medication | Support | 6-piece (team-unique): Black Ink gain -8 at battle end; at battle start, add original versions of wearer's non-strike/non-defense cards to draw pile (max 1 per type). | — |
| Ring of 36 Chambers | DPS | 6-piece (team-unique): Silver Key cap and skill cost +200; Overload crit rate, crit DMG, shield, heal +30%. | — |
| Further Metamorphosis | — | Covenant set mentioned in English strategy guides. | — |
| Apocalypse | Endgame | Endgame content covenant set. | — |

## Main Stat Options

Covenant pieces can roll the following main stats:

- Crit Rate
- Crit DMG
- DMG Amplification
- Keyflare Regen (Silver Key)
- Aliemus Regen
- Realm Mastery
- Death Resist
- Black Ink Drop

::: tip
Covenants can be equipped by any Awakener. Focus on farming Covenants with stats that match your main team's needs. Covenant transcription costs 3 Thousand-Face Seals + 7,500 Rose Scrip per attempt.
:::
