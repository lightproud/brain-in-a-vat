# Consumables

## Experience Materials

Used to level up Awakeners by granting experience points.

| Item Name | Rarity | Effect | How to Obtain |
|-----------|--------|--------|---------------|
| Gnosis Elixir I (灵知制剂 I) | Common (Green) | Grants a small amount of EXP. Overflow returns lowest-tier elixirs (rounded to 500) | Erosion Ruins daily dungeons |
| Gnosis Elixir II (灵知制剂 II) | Rare (Blue) | Grants a moderate amount of EXP | Erosion Ruins daily dungeons |
| Gnosis Elixir III (灵知制剂 III) | Epic (Purple) | Grants a large amount of EXP | Erosion Ruins daily dungeons |
| Special Approval Elixir (特批药剂) | High-tier | Grants a large amount of EXP | Investigation chapter rewards (Ch.1 = 5, Ch.6 = 35) |

## Universal Materials

Multi-purpose materials used across various progression systems.

| Item Name | Effect | How to Obtain |
|-----------|--------|---------------|
| Mimetic Crystal (拟态凝晶) | Universal synthesis resource. 30 = 1 advanced skill material. Breakthrough materials return at 30:1 on recall | Curriculum, Sediment/Deep Dive/Rose Scrip shops |
| Memory Quill (追念羽笔) | Universal upgrade material (unlocked at level 55). Buy 25-50 per week from 5 different shops | Investigation Badge, Erosion Pearl, Lightless, Phase, Sediment shops |
| Thousand-Face Seal (千面印章) | Covenant enhancement and transcription material. 36 seals to enhance a Covenant from +0 to +12 | Curriculum, events |
| Primordial Spirit Core (原初灵核) | Gnosis Deepening material (level cap 60 to 70). Each deepening requires 1 core + domain materials + Rose Scrip | Sediment shop (25 shards = 1 core, 3/month) |

## Roguelike Relics

Relics found during roguelike dungeon runs. These are **temporary** and only last for the current exploration.

| Tier | Cost | Description |
|------|------|-------------|
| Silver | ~50 Black Sigils | Moderate effects. Available in shops |
| Gold | ~90 Black Sigils | Powerful effects. Available in shops |
| Cursed | N/A | Powerful effects with drawbacks. Only from elite enemies and special events; not sold in shops |
