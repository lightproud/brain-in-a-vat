# Currency

Morimens features several types of in-game currency used for summoning, purchasing items, and upgrading.

## Premium Currencies

| Currency | Usage | How to Obtain |
|----------|-------|---------------|
| Silver Prime | Paid premium currency. Used to purchase summon items, battle pass, cosmetics, and gift boxes. | Real-money purchase |
| Silver (Silver Core) | Primary free premium currency. Exchange for Luminous Core or Pure Core in the shop. Can also restore Menophine (50 Silver = 120 Menophine). | Daily tasks, main story scores, main story achievements, handbook, Phantasmal Dive, events, battle pass, redemption codes, map exploration |

## Summoning Currencies

| Currency | Usage | How to Obtain |
|----------|-------|---------------|
| Luminous Core | Limited banner summon item. Used for limited character and Wheel of Destiny banners. Priority exchange recommended. | Silver exchange, events, battle pass |
| Pure Core | Standard banner summon item. Used for standard character and Wheel of Destiny banners. More sources available than Luminous Core. | Silver exchange, events, first Awakener ascension reward, battle pass free track (40x), chapter first clear |

## Exchange Currencies

| Currency | Usage | How to Obtain |
|----------|-------|---------------|
| Rose Scrip | Main general currency. Used in Rose Scrip Shop, edification, alchemy, material synthesis, wheel stacking, covenant transcription (7,500/attempt). | Main story first clear, Dissolution Ruins, Outfield Agent, Curriculum, wheel disassembly (Blue=1,000, Purple=4,000) |
| Sediment Currency | Used in Sediment Shop. Purchase covenant sets (Scarlet Embrace), Memory Quill (80 sediment = 10 quills/week), Primordial Spirit Core (25 shards = 1 core, 3x/month). | Summoning (gacha byproduct) |
| Investigation Badge | Used in Investigation Shop. Purchase Memory Quill (800 badges = 5 quills/week). | Main story 3-star clear, Arcane Dominion, Soul Dive 3-star, True Mastery (1,000), personal achievements (300 each) |
| Lightless Currency | Used in Lightless Shop. Purchase covenant sets (Distant Feast), Memory Quill (800 = 5/week). | Lightless Realm, D-Effect Zone |
| D-Jewels (Erosion Pearls) | Used in Deep Dive Shop. Purchase Memory Quill (800 = 5/week), Shattered Remnant Dream (100/week), Stellar Remnant Dream (25/week). Formula: (Phantasmal level + 2) x 100. | Phantasmal Dive (Level 3 = 500, Frenzy = 900), Outfield Agent (daily settlement) |
| Phase Coin | Used in Phase Shop. Purchase covenant sets (Scarlet Palpitation), Memory Quill (800 = 5/week). | Phase Chess (PVP) |

## Stamina

| Currency | Usage | How to Obtain |
|----------|-------|---------------|
| Menophine | Stamina/action resource consumed when entering intermission drills. 50 Silver = 120 Menophine. | Natural recovery, Silver exchange, items/gift boxes |

## Currency Spending Advice

1. **Save premium currency for limited banners** -- Do not spend on the standard banner unless you have surplus.
2. **Prioritize Luminous Core** -- When exchanging Silver, Luminous Core should be prioritized over Pure Core since limited banner characters are time-restricted.
3. **Buy stamina refreshes wisely** -- The first few daily stamina refreshes are cost-efficient; later ones are not.
4. **Clear event shops** -- Event currency expires when the event ends. Spend it before the deadline.
5. **Prioritize summoning resources** -- In most shops, summon tickets and pulls offer the best long-term value.

::: warning
Premium currency spent on the standard banner is generally less efficient than saving for a limited banner with a specific Awakener you want.
:::
