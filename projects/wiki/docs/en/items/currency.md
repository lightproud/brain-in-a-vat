# Currency

Morimens features several types of in-game currency used for summoning, purchasing items, and upgrading.

## Currency Types

| Currency | Usage | How to Obtain |
|----------|-------|---------------|
| *Premium Currency* | Gacha pulls, shop purchases | Events, main story, daily tasks, real-money purchase |
| *Standard Currency* | General purchases, upgrades | Stage rewards, daily tasks, Dispatch |
| *Arena Currency* | PvP shop purchases | Phase Chess rewards |
| *Event Currency* | Event shop purchases | Event participation |

## Misag Shop

The **Misag Shop** is the primary in-game store where players spend various currencies.

| Shop Section | Currency Accepted | Items Available |
|-------------|-------------------|-----------------|
| General | Standard currency | Materials, EXP items |
| Premium | Premium currency | Summon tickets, rare materials |
| Event | Event currency | Limited-time items, Awakener shards |
| Arena | Arena currency | Covenants, Engravings |

::: tip
Check the Misag Shop refresh for bargains. Some items in the shop are significantly more stamina-efficient than farming them from stages.
:::

## Currency Spending Advice

1. **Save premium currency for limited banners** -- Do not spend on the standard banner unless you have surplus.
2. **Buy stamina refreshes wisely** -- The first few daily stamina refreshes are cost-efficient; later ones are not.
3. **Clear event shops** -- Event currency expires when the event ends. Spend it before the deadline.
4. **Prioritize summoning resources** -- In most shops, summon tickets and pulls offer the best long-term value.

::: warning
Premium currency spent on the standard banner is generally less efficient than saving for a limited banner with a specific Awakener you want.
:::
