# Resource Stages

Resource stages are the primary source of upgrade materials in Morimens. They are available daily and consume Menophine (stamina).

## Resource Stage Types

| Stage | Type | Reset | Unlock | Drops |
|-------|------|-------|--------|-------|
| Dissolution Ruins | Material Dungeon | Daily | Clear Investigation 1-2 | Ascension Caskets, Skill Supplies, Domain Residues, Edification Materials, Rose Scrip, Gnosis Elixirs, Breakthrough Materials |
| Forbidden Inscriptions | Material Dungeon | Daily | Post-clear repeat | Various upgrade materials |
| Transcendent Existence | Weekly Boss | Weekly (Monday 15:00) | Story progression | Rich rewards -- clear weekly for best value |

## Dissolution Ruins Details

The Dissolution Ruins are the main material farming location. They consume Menophine (stamina) to enter and contain two subtypes:

- **Ascension Material Runs** -- Drop Ascension Caskets and Breakthrough Materials
- **Skill Material Runs** -- Drop Skill Supplies and Domain Residues

Each realm has its own Dissolution Ruins variant:

| Realm | Dungeon Name | Exclusive Drop |
|-------|-------------|----------------|
| <span class="realm-badge realm-chaos">Chaos</span> | Chaos Dissolution Ruins | Chaotic Residue |
| <span class="realm-badge realm-aequor">Aequor</span> | Aequor Dissolution Ruins | Abyssal Residue |
| <span class="realm-badge realm-caro">Caro</span> | Caro Dissolution Ruins | Bloody Residue |
| <span class="realm-badge realm-ultra">Ultra</span> | Ultra Dissolution Ruins | Cosmic Residue |

## Daily / Weekly System

| Activity | Reset | Rewards |
|----------|-------|---------|
| Daily Training | Daily | 500 EXP/day. Can use proxy certificates for quick completion (max 5x). |
| Weekly Trial | Weekly | Silver 100/150/250 + D-Jewels |
| Outfield Agent (Assignments) | Daily settlement | Rose Scrip, D-Jewels, various materials. Research: Recipe compendium tasks yield 4,000 base + 2,000 per 20-level gap Rose Scrip. |
| Curriculum | Ongoing | Rose Scrip, Mimetic Crystal, Thousand-Face Seals |

## Farming Priority

1. **Skill materials** -- Skill levels have the biggest impact on combat performance.
2. **Awakener EXP** -- Level your core team to at least 25 for skill unlocks.
3. **Ascension materials** -- Required to break level caps.
4. **Edification materials** -- For character trait upgrades.
5. **Clear Transcendent Existence weekly** -- Rich rewards that should not be missed.

::: tip
All main story stages have 3-star rating rewards worth 50 Silver each, totaling up to 4,750 Silver across all stages. Clearing a higher difficulty automatically unlocks lower difficulty 3-star rewards.
:::
