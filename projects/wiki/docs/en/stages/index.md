# Stages

Morimens features multiple stage categories, each offering different challenges and rewards.

## Stage Types

| Type | Description | Stamina | Link |
|------|-------------|---------|------|
| Main Story | Progress the narrative across 9+ chapters | Yes | [Main Stages](./main.md) |
| Resource | Farm upgrade materials on a daily rotation | Yes | [Resource Stages](./resource.md) |
| Challenge | High-difficulty encounters with premium rewards | Limited | [Challenge Stages](./challenge.md) |

## Progression Path

1. **Main Story** -- Clear story chapters to unlock game features and modes.
2. **Resource Stages** -- Farm materials to level and upgrade your Awakeners and cards.
3. **Challenge Stages** -- Test your team's strength for the best rewards.

::: tip
If you are stuck on a main story stage, spend stamina on resource stages to strengthen your team before retrying.
:::
