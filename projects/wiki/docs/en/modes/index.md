# Game Modes

Morimens offers a variety of game modes spanning combat, exploration, PvP, crafting, and more.

## Mode Overview

| Mode | Type | Description | Unlock |

| [Investigation](./investigation.md) | PvE | Roguelite hex-map exploration | Main Story |
