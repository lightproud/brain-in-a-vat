# Worldbuilding

Morimens is set in a world deeply influenced by the **Cthulhu Mythos** and Lovecraftian horror. This page catalogues the key concepts, factions, and lore elements of the game's world.

## Cthulhu Mythos Concepts

The game draws from and reinterprets classic Lovecraftian concepts:

| Game Concept | Mythos Inspiration | Role in Morimens |
|-------------|-------------------|-----------------|
| Awakeners (唤醒体) | Awakened humans | Playable characters with eldritch powers |
| Designated Realms | Domains of cosmic entities | Four schools of power (Chaos, Aequor, Caro, Ultra) |
| Madness Points (狂气) | Sanity loss | Combat mechanic; triggers Madness Burst at 100 |
| Silver Key | The Silver Key (Lovecraft) | Combat energy; activates Key Orders at 1000 |
| The Keeper | Investigator / Curator | Player character; protector of Awakeners |
| Night of Oblivion (忘却前夜) | Cosmic apocalypse | The world-ending event the story revolves around |
| Investigation | Mythos investigation | Roguelite exploration mode |
| Tentacles (Aequor) | Deep Ones / Cthulhu | Summoned entities in combat |
| Embryos (Caro) | Shoggoths / flesh horror | Created and consumed cards |

## The World

### The Night of Oblivion

The "Night of Oblivion" (忘却前夜) is the central catastrophe of Morimens' world. It is a cyclical event that threatens to erase memory, identity, and ultimately existence itself. The Keeper and the Awakeners stand against this tide.

## The Four Realms as Cosmic Domains

Each Designated Realm represents a different facet of the unknowable:

- <span class="realm-badge realm-chaos">Chaos</span> -- The primordial void from which all things emerged. Represents formlessness and infinite potential.
- <span class="realm-badge realm-aequor">Aequor</span> -- The abyssal deep, where ancient entities slumber beneath impossible pressures.
- <span class="realm-badge realm-caro">Caro</span> -- The flesh that reshapes itself, evoking body horror and mutation beyond natural law.
- <span class="realm-badge realm-ultra">Ultra</span> -- The dimensions beyond human perception, where time and space lose meaning.
