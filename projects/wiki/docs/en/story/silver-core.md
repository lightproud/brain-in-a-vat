# Silver Core Communication

**Silver Core Communication** is Morimens' affinity system. It allows the Keeper to build relationships with Awakeners through interactions, gifts, and shared experiences, unlocking story content and gameplay bonuses.

## Overview

| Aspect | Detail |
|--------|--------|
| System Type | Affinity / Bond system |
| Purpose | Unlock character stories, voice lines, stat bonuses |
| Progression | Raise affinity levels through various activities |

## How It Works

1. **Interact** with Awakeners through the Silver Core Communication interface.
2. **Give gifts** -- Each Awakener has preferred and disliked gifts.
3. **Include in battles** -- Using an Awakener in combat slowly builds affinity.
4. **Complete character quests** -- Special quests that deepen the bond.

Each affinity level unlocks new content:

- **Story Episodes** -- Personal backstory chapters for the Awakener.
- **Voice Lines** -- New dialogue and voice clips.
- **Stat Bonuses** -- Minor stat increases for the Awakener.
- **Special Content** -- Expressions, profile items, or other cosmetics.

## Gift Giving

| Gift Preference | Affinity Gain |
|----------------|---------------|
| Preferred | High |
| Neutral | Low |
| Disliked | Negative or none |

::: tip
Pay attention to each Awakener's personality and lore to guess their gift preferences. Preferred gifts grant significantly more affinity than neutral ones.
:::

## Tips

- **Interact daily** -- Small daily interactions add up over time.
- **Focus on your main team** -- Prioritize affinity for Awakeners you use in combat to gain their stat bonuses.
- **Read the stories** -- Character stories unlocked through Silver Core Communication provide valuable lore context.
