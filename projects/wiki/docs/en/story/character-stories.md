# Character Stories

Each Awakener in Morimens has a personal backstory that can be unlocked by raising their affinity through the [Silver Core Communication](./silver-core.md) system.

## How to Unlock Character Stories

1. Raise an Awakener's affinity level through Silver Core Communication.
2. Reach specific affinity thresholds to unlock story episodes.
3. View unlocked stories from the Awakener's profile page.
