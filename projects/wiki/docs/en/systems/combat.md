# Combat System

## Combat Basics

Morimens uses a real-time combat system where players can control characters to attack, use skills, and dodge.

### Basic Actions

| Action | Description |
|--------|-------------|
| Normal Attack | Tap repeatedly for combo attacks |
| Skill | Consume skill points to cast character skills |
| Dodge | Evade enemy attacks; a perfect dodge triggers a time slowdown |
| Switch Character | Swap the active battle character |
| Ultimate | Unleash a powerful ultimate skill when the energy bar is full |

### Elemental Counters

Elemental advantage is critical in combat and deals bonus damage against the weak element.

## Damage Calculation

### Damage Formula

Base damage calculation involves the following factors:

- ATK
- Skill multiplier
- Elemental advantage bonus
- Crit DMG
- DEF reduction
- Damage increase / decrease buffs

### Crit System

| Stat | Description |
|------|-------------|
| Crit Rate | Probability of landing a critical hit |
| Crit DMG | Damage multiplier on crit, base 150% |

## Combat Tips

1. **Master dodging** - A perfect dodge grants bonus action opportunities
2. **Chain skills together** - Some character skills create combo effects when linked
3. **Time your switches** - Switching characters triggers on-entry effects
4. **Target elemental weaknesses** - Build your team around the enemy's weak element
