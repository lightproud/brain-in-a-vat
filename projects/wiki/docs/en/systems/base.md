# Base Building System

## Overview

The base serves as the player's headquarters, where various facilities can be constructed to generate resources and enhance characters.

## Key Facilities

The base in Morimens is centered around **Mythag University** (弥萨格大学), located in the Kingdom of Leo. Rather than a traditional building system, the base provides access to various functional facilities that support daily progression.

| Facility | Function | Details |
|----------|----------|---------|
| **Smelting Room** (冶炼室) | Equipment/item exchange | Exchange and craft items, gear, and materials |
| **Assignments** (外勤代理) | Auto-dispatch system | Send Awakeners on assignments that reward Rose Scrip, Erosion Pearls, and various materials daily |
| **Curriculum** (课程) | Advanced training | Complete courses to earn Mimetic Crystals, Thousand-Face Seals, and Rose Scrip |
| **Daily Training** (每日训练) | Daily tasks | Merged daily + weekly tasks; can use proxy vouchers for quick completion (up to 5 times). Grants 500 EXP/day |
| **Weekly Trial** (每周试炼) | Weekly challenges | Weekly challenge missions awarding 100/150/250 Silver and Erosion Pearls |
| **Alchemy** (炼金) | Material synthesis | Combine 3 lower-tier materials into 1 higher-tier material; 30 Mimetic Crystals = 1 advanced skill material |

## Assignments (Outfield Agent)

The Assignments system allows you to dispatch Awakeners on automated missions. Rewards are settled daily and include:

- **Rose Scrip** (蔷薇金券) -- The primary universal currency
- **Erosion Pearls** (融蚀之珠) -- Used in the Deep Dive shop
- **Various materials** -- Ascension caskets, skill supplies, and more

::: tip
The "Recipe Compendium" assignment type grants 4,000 base Rose Scrip plus an additional 2,000 for every 20 levels of difficulty difference. Prioritize these for maximum currency income.
:::

## Curriculum

The Curriculum system offers advanced courses that provide core progression materials:

- **Mimetic Crystals** (拟态凝晶) -- Universal synthesis resource (30 = 1 advanced skill material)
- **Thousand-Face Seals** (千面印章) -- Covenant enhancement and transcription material
- **Rose Scrip** -- Additional currency income

## Resource Tips

1. **Check Assignments daily** -- Rewards settle automatically each day; reassign Awakeners to maximize output
2. **Complete Daily Training** -- Use proxy vouchers to save time if available
3. **Run Curriculum regularly** -- Mimetic Crystals are essential for late-game material synthesis
4. **Visit the Smelting Room** -- New recipes unlock as you progress through the main story
