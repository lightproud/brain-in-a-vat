# Base Building System

## Overview

The base serves as the player's headquarters, where various facilities can be constructed to generate resources and enhance characters.

## Resource Production

Base buildings can automatically generate various resources:

- Currency
- EXP materials
- Upgrade materials

## Construction Tips

1. **Prioritize production buildings** - Ensure a stable resource income
2. **Develop evenly** - Avoid over-investing in a single building type
3. **Collect resources on time** - Resources have a storage cap and need to be collected regularly
