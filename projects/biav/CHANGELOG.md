# 缸中之脑终端 更新日志

> BIAV Terminal — AI 增强对话终端，忘却前夜（Morimens）项目配套工具

---

## v0.15.0 — 团队基座 · 网关模式 · 数据隔离
**2026-04-05**

- 🌐 **API 网关模式**：支持配置内部 API 网关地址 + Token，团队部署无需暴露原始 API Key
- 🔀 **统一 API 层**：所有 4 处 Anthropic 调用（主对话/建议/后台任务/自动摘要）统一走 `getApiConfig()` + `buildApiHeaders()`
- 👥 **用户数据隔离**：对话和记忆按用户名分别存储（`biav-{user}-convs` / `biav-{user}-memory`），切换账号自动加载对应数据
- ⚙️ **网关设置 UI**：设置面板新增 API 网关 URL + 网关 Token 输入，附说明文字
- 🔄 **账号切换刷新**：登录/切换/退出账号时自动重载对话列表和首页状态

## v0.14.0 — Agent 架构升级 · Claude Code 模式
**2026-04-05**

- ⚡ **MicroCompact 裁剪**：旧工具输出自动截断，旧消息裁剪，零 API 调用节省 token
- 🗜 **AutoCompact 摘要**：对话超 20 轮/40K 字符时用 Haiku 压缩前半段为摘要
- 🧠 **记忆索引化**：自动分类（用户偏好/项目状态/游戏知识/技术笔记/决策记录），索引始终注入、详情按需加载
- ↩️ **转向队列**：流式输出期间可直接发送新消息，自动中断当前流并切换方向
- 🔒 **PreToolUse Hook**：安全模式下高风险工具（JS 执行、AI 绘图）执行前弹出确认
- 😤 **情绪感知**：正则检测用户负面情绪，自动在提示词中追加耐心指令
- 🤖 **子 Agent 后台任务**：`background_task` 工具用 Haiku 异步执行摘要/翻译等任务，主对话不阻塞

**2026-04-05**
