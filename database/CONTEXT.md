# 官方数据库 — 子项目上下文

> 本文件供"官方数据库"会话使用，启动时请先阅读根目录 `CLAUDE.md` 了解全局。

## 职责

系统性整理忘却前夜（Morimens）的官方游戏数据，为衍生游戏开发提供数据基础。

## 目标数据范围

- 角色信息（属性、立绘、背景故事）
- 技能参数（数值、效果、冷却）
- 武器/装备数据
- 关卡/副本信息
- 游戏机制（战斗系统、元素反应等）
